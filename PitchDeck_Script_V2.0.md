# BDS 소상공인 플랫폼: 최종 투자자 피치 스크립트 초안 (V2.0)

## 📌 목표 청중 및 목적
*   **청중:** 초기 단계의 외부 파트너, 잠재적 투자사(VC).
*   **목표:** '기술력'을 넘어선 **'시장 성공 가능성'과 '사회적 임팩트를 통한 수익화 구조'**를 증명하는 것.

## 🎤 스크립트 구성 원칙 (전체 가이드)
1.  **톤앤매너:** 자신감 있고, 문제의 심각성을 강조하며(Pain Point), 해결책을 제시할 때는 희망적이고 구체적인 어조 유지.
2.  **핵심 메시지 반복:** "기술은 안전장치이며, 이 안전장치가 곧 새로운 수익 구조를 만든다"는 메시지를 모든 섹션에 녹여낼 것.

*   **[솔루션 제시]**: "저희 플랫폼은 단순 결제가 아닙니다. 사장님의 비즈니스를 24시간 지키는 **'AI 기반의 안전망(Safety Net)'**입니다." (기술적 안정성을 감성적으로 전환)
