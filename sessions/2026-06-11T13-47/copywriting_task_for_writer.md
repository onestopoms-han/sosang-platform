# ✍️ 안전마진 메시지 카피라이팅 작업 지시

**Researcher 에서 작성한 보고서 초안 기반의 '안전마진' 메시지에 대한 구체적인 카피라이팅 작업을 요청합니다.**

## 1. 작업 배경 및 목표
CEO 가 지시하신 '안전마진'이라는 개념을 데이터 시각화 (PainGauge, TrustWidget 등) 와 어떻게 연결하여 사용자의 감성적 경험에 성공적으로 전달할지 분석했습니다. 이제 이 분석 결과를 바탕으로, 사용자가 플랫폼을 접했을 때 느끼는 '안정감'과 '신뢰감'을 자극하는 구체적인 카피를 작성해야 합니다.

## 2. 핵심 메시지 (Key Message)
- **안전마진:** "당신의 상태를 안정화시켜드립니다" 또는 "불안을 해소하고 안정된 상태로 전환해드립니다"
- **PainGauge + TrustWidget:** "불안정함이 안정된 상태로 전환됩니다" 또는 "신뢰도 점수가 상승합니다"
- **BDS 플랫폼:** "소상공인의 어려움에 공감하며, 실질적인 도움을 제공합니다"

## 3. 카피라이팅 작업 항목 (Writer 에게 전달할 구체적인 지시)
1. **온보딩 시나리오 카피:** 사용자가 PainGauge 을 처음 접했을 때, '안전마진'이라는 개념을 이해하게 하는 메시지 작성.
2. **진단 결과 보고 화면 카피:** PainGauge 에서 안전 구간으로 진입하고 TrustWidget 점수가 상승할 때, 사용자에게 전달하는 메시지 작성.
3. **행동 유도 (CTA) 카피:** 데이터 시각화를 통해 '안전마진'을 경험한 후, 사용자가 다음 행동을 취하도록 유도하는 메시지 작성.
