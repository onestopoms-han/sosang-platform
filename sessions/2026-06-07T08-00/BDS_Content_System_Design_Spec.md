# BDS Content System Design Spec v1.0
## 1. 핵심 컨셉: 안전마진 (Safety Margin)
> "불확실한 미래에 대비하는 소상공인의 지혜를, 데이터로 시각화합니다."

## 2. 색감 및 톤앤매너
- **메인 컬러:** 신뢰의 파란색 (#0F4C75) — 안정감, 전문성
- **강조 컬러:** 경고의 주황색 (#E07A5F) — 위험도, 주의 필요 (PoC 에서의 🔴→🟡→🟢 변화와 연결)
- **배경:** 따뜻한 오프화이트 (#FDFCF5) — 골목상권의 따뜻한 느낌

## 3. 비주얼 요소 (Visual Assets)
- **손바닥 손가락 표시 아이콘:** 손에 숫자 '5'가 떠 있는 디자인, 직관적 이해도 확보
- **손전등 효과:** 어둠 속에서 빛이 나며 위험을 경고하는 시각적 동기 부여
- **손전조명:** 어두운 골목상권의 현실을 비추는 빛, 데이터 기반의 해답 제시

## 4. 브랜드 스타일
- **타겟:** 소상공인 (30~60 대) — 직관적이고 명확한 정보 전달
- **톤:** 진지하고 따뜻한 목소리 — 공감과 신뢰를 동시에 표현
