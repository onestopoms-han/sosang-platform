# 🎬 BDS소상공인플렛폼 Reels Voiceover Script (Week 1)

## [R1: Hook + PainGauge 시각화]  
**Duration:** 60 초  
**Tone:** 차분함, 경고감 → 희망  

| Time | Visual Guide | Voiceover (내레이션) | Subtitle/On-Screen Text |
|------|--------------|---------------------|------------------------|
| **0~5s** | 가게 내 혼잡한 모습 (Red tint). 안전마진 경고음 효과. | "당신의 가게, 안전마진이 있나요?" | [자막] 당신의 가게는 안전마진이 있나요? |
| **5~20s** | PainGauge 애니메이션 등장: Red 영역 → Green 영역 타임랩스. | "운영하다 보면 손실은 빨라지고, 수익은 느려집니다." | [자막] Red=손실 / Green=수익 (타임랩스) |
| **20~35s** |店主의 눈빛 클로즈업: PainGauge 숫자 변화 (손실 → 안정). | "PainGauge 이 당신의 가게 상태를 실시간으로 보여줍니다." | [자막] Red→Yellow→Green, 작은 신호가 큰 결정을 만듭니다. |
| **35~50s** | Green 영역이 확대되며 '안전' 아이콘 등장. | "Red 영역을 줄이면, 수익 기회가 열립니다." | [자막] 안전마진, 지금 확인하세요! |
| **50~60s** | CTA 버튼: '안전마진 확인하기' (Trust Widget 로직 적용). | "지금 바로 PainGauge 으로 무료 진단 받으세요!" | [자막] 👇 지금 무료로 진단받으세요! |

## [R2: Voiceover + Loss Gauge 설명]  
**Duration:** 75 초  
**Tone:** 차분함, 분석적  

| Time | Visual Guide | Voiceover (내레이션) | Subtitle/On-Screen Text |
|------|--------------|---------------------|------------------------|
| **0~15s** | 손실 비용 흐름 차트: Red → Yellow → Green 색상 전환. | "PainGauge 은 손실 비용을 시각화합니다." | [자막] 손실 비용 = Red 영역 (위험) |
| **15~30s** | 실제 데이터 애니메이션이 나옴. Voiceover가 PainGauge 로직 설명. | "Red 영역은 위험 신호, Green 영역은 수익 기회입니다." | [자막] Red=위험 / Yellow=주의 / Green=기회 |
| **30~45s** | Trust Widget 예시: 신뢰도 0.3 → 0.7 임계값 전환. | "신뢰도 0.7 을 넘으면, 안전마진이 활성화됩니다." | [자막] Trust Widget: 신뢰도 0.7 → 안전마진 ON |
| **45~60s** | Green 영역이 확대되며 '무료 체험' 버튼 등장. | "지금 바로 PainGauge 으로 무료 진단 받으세요!" | [자막] 📲 지금 무료로 진단받으세요! |
| **60~75s** | CTA: 'PainGauge 무료 체험 시작'. | "PainGauge, 당신의 가게를 위해 있습니다." | [자막] PainGauge 무료 체험 시작 |

## [R3: Subtitle Focus (Text-Driven)]  
**Duration:** 45 초  
**Tone:** 간결함, 강렬함  

| Time | Visual Guide | Voiceover (내레이션) | Subtitle/On-Screen Text |
|------|--------------|---------------------|------------------------|
| **0~10s** | 텍스트 중심: "당신의 가게는 안전마진이 있나요?" | (무음, 배경 음악만) | [자막] 당신의 가게는 안전마진이 있나요? |
| **10~25s** | PainGauge 애니메이션: Red → Green 전환. | (짧은 VO: "Red 영역 줄이세요!") | [자막] Red=손실 / Green=수익 (PainGauge) |
| **25~45s** | CTA: '안전마진 확인하기' 버튼 등장. | "지금 바로 확인하세요!" | [자막] 📲 지금 무료로 진단받으세요! |
