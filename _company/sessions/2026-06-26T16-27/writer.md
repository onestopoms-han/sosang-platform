19	## 2. 기술적 제약 사항 (Technical Constraints)
20	
21	### 2.1 API 응답 시간 및 데이터 무결성
22	- **API 응답 시간:** 모든 엔드포인트는 `50ms` 이내 응답해야 합니다. 지연이 발생하면 `PainGauge` 컴포넌트는 로딩 상태를 표시합니다.
23	- **데이터 무결성:** `Trust Widget` 의 데이터를 변경하려면 반드시 `DB` 를 업데이트하고, UI 는 실시간으로 동기화되어야 합니다.
24	- **보안 정책:** 모든 API 요청은 HTTPS 가 필수이며, 민감한 데이터는 암호화된 형태로 전송됩니다.
