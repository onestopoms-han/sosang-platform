## ✅ MVP 핵심 기능 3 가지 (API 불확실성 대응 버전)

### 1️⃣ **송이버섯 판매 플랫폼 — 기본 거래 시스템**  
- **기능**: 소상공인(판매자) ↔ 소비자(구매자) 간 송이버섯 거래 매칭
- **MVP 필수**: 상품 등록, 주문 처리, 결제 연동 (PayPal 연동 시도 중)
- **우선순위**: 🔴 최상 (기반 인프라)

### 2️⃣ **Trust Widget 및 PainGauge — 신뢰·성장 시각화**  
- **기능**: 판매자 신뢰도 점수 (Deep Blue), 성장 지표 (Green/Yellow/Red)
- **MVP 필수**: API 없이도 HTML/JS 로 구현 가능 (코다리가 코드 작성 중)
- **우선순위**: 🟡 2 위 (신뢰 메커니즘)

### 3️⃣ **Pilot 데이터 수집 프로세스 — 최소 기술 요구**  
- **기능**: 판매자 기본 정보, 초기 송이버섯 재고, 고객 피드백
- **MVP 필수**: 구글 폼/엑셀 또는 간단한 API 로 대체 가능 (코다리가 파일 접근 실패 시)
- **우선순위**: 🟢 3 위 (데이터 기반 개선)

# 🧪 Pilot 데이터 수집 프로세스 — 최소 기술 요구 버전

## 🎯 목표
API 스키마 불확실성 하에서도 **즉시 실행 가능**한 Pilot 그룹 데이터 수집 프로세스를 제공합니다.  
기존 API 기반 설계를 구글 폼/엑셀/유튜브 댓글 등 대체 수단으로 전환합니다.

## ✅ 단계별 작업 (기술적 제약 대응)

### 1️⃣ 판매자 기본 정보 수집 — 구글 폼
- **구현**: [Google Forms](https://forms.google.com/) + Google Sheets 연동
- **필요 데이터**:  
  - 소상공인 이름, 연락처, 사업자 등록번호
  - 송이버섯 재고량 (kg), 평균 가격 (원/kg)
  - 공급처 정보 (산지, 유통 경로)
- **장점**: 별도 개발 없이 5 분 내 생성 가능, 자동 저장

### 2️⃣ 고객 피드백 수집 — 유튜브 댓글 + 인스타그램 DM
- **구현**: 레오/Instagram 에이전트가 이미 운영 중인 채널 활용
- **필요 데이터**:  
  - 구매자 만족도 (1~5 점), 제품 품질 평가
  - 개선 제안 (가격, 배송 속도 등)
- **장점**: API 없이도 텍스트 기반 피드백 수집 가능

### 3️⃣ 재고 및 판매 데이터 — 엑셀/CSV 파일
- **구현**: 코다리가 개발하는 기본 거래 시스템과 연동하지 않고, 독립적인 CSV 로그 유지
- **필요 데이터**:  
  - 매일의 판매량 (kg), 평균 매출액 (원)
  - 반품률, 고객 문의 빈도
- **장점**: API 없이도 파일 기반 저장 가능

## 📊 비용 대비 효과 분석

| 수집 수단 | 개발 시간 | 예상 데이터 품질 | MVP 필수 여부 |
|-----------|-----------|------------------|---------------|
| 구글 폼/시트 | 0 시간 (설치) | ⭐⭐⭐⭐ (자동화) | ✅ 필수 |

| 엑셀/CSV 로그 | 5 분 (구글 스프레드시트) | ⭐⭐⭐⭐ (수동 입력) | ✅ 필수 |
