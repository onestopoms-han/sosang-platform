## 🌟 BDS소상공인플랫폼 최종 비주얼 가이드라인 (V3.0 확정)

### 1. 브랜드 컬러 시스템 (Brand Color System)
신뢰도와 위기감을 표현하기 위한 핵심 색상 팔레트입니다. 모든 마케팅 콘텐츠에서 이 조합을 최우선으로 사용합니다.

| 용도 | 색상 코드 (HEX) | 역할 및 의미 | 적용 예시 |
| :--- | :--- | :--- | :--- |
| **Primary (신뢰/안정)** | `#007BFF` | 플랫폼의 신뢰성과 안정성을 나타냄. 핵심 CTA 버튼에 사용. | 버튼, 주요 헤드라인 배경 |
| **Secondary (긍정/성장)** | `#28A745` | 긍정적인 결과와 성장을 강조. 성공 지표 표시. | KPI 달성, 긍정적 피드백 |
| **Warning (위기/주의)** | `#FFC107` | 잠재적 위험 또는 주의가 필요한 지점 강조. | PainGauge의 경고 영역 |
| **Danger (위험/긴급)** | `#DC3545` | 심각한 위기 상태, 즉각적인 조치가 필요함을 알림. | Trust Widget의 위험 수치 |
| **Background (신뢰 기반)** | `#E6F7FF` | 전체 콘텐츠 배경으로 사용되어 신뢰감을 높임. | 포스트/썸네일 배경 |
| **Neutral (텍스트)** | `#343A40` | 본문 텍스트 및 주요 정보 전달. | 제목, 설명 텍스트 |

### 2. 타이포그래피 시스템 (Typography System)
가독성과 브랜드 톤을 유지하기 위한 폰트 사용 규칙입니다.

*   **Primary Font (제목/강조):** **Pretendard Bold / SemiBold** (최대 시각적 임팩트를 위해 사용)
    *   사용처: 헤드라인, 핵심 KPI 숫자, CTA 버튼 텍스트
*   **Secondary Font (본문/설명):** **Pretendard Regular**
    *   사용처: 상세 설명, 본문 내용, 플랫폼 소개
*   **Hierarchy Rule:** 모든 콘텐츠는 Primary Font를 사용하여 시선을 집중시키고, Secondary Font로 세부 정보를 명확히 구분합니다.

### 3. Trust Widget / PainGauge 비주얼 명세 (핵심 컴포넌트)
이 컴포넌트의 시각적 표현은 **색상 시스템**을 통해 감정적 반응을 유도하는 데 중점을 둡니다.

*   **Trust Widget (신뢰도 측정):**
    *   **Full Trust:** `#28A745` (녹색 계열) - 안전하고 성공적인 느낌.
    *   **Medium Trust:** `#007BFF` (파란색 계열) - 일반적인 신뢰 범위.
    *   **Low Trust:** `#FFC107` (노란색 계열) - 주의 필요함을 알림.
    *   **Critical Risk:** `#DC3545` (빨간색 계열) - 즉각적인 위기 상황 강조.
*   **PainGauge (고통/어려움 측정):**
    *   측정 수치에 따라 **Danger 색상(#DC3545)**의 채도나 투명도를 조절하여 시각적 압박감을 조성합니다.

### 4. 썸네일/포스트 표준 레이아웃 (Template Standardization)

마케팅 콘텐츠 제작 시 다음 3가지 표준 템플릿을 즉시 적용합니다. 모든 템플릿은 **Background: `#E6F7FF`**를 기본으로 합니다.
