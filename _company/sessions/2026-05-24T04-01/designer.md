Loss Gauge 시각화와 Red/Green 시스템이 '경제적 생존 전략'이라는 스토리텔링을 가장 명확하고 매력적으로 전달하도록 최종 UI 컴포넌트의 상태(State) 검증 및 미세 조정을 완료합니다.

## 1. 핵심 디자인 원칙 확정 (Storytelling & Color Mapping)

**핵심 메시지:** 소상공인의 공포(Loss)를 구체적인 행동 지침(Survival Strategy)으로 전환한다.

| 시스템 | 색상 코드 | 심리적 의미 (스토리텔링) | 적용 레이아웃/UI 요소 |
| :--- | :--- | :--- | :--- |
| **위험 영역 (Loss)** | `#E74C3C` (Red/Alert) 또는 `Deep Blue`의 명암 대비를 활용한 강렬한 빨강 계열 | 즉각적인 위험, 손실, 경고 (생존 위협) | Loss Gauge 바 자체의 채움 색상, 경고 텍스트 |
| **안정 영역 (Gain)** | `#2ECC71` (Growth Green) | 성장, 안전, 기회, 생존 전략 (성장 기회) | Gain Gauge 바 자체의 채움 색상, 성공 메시지 |
| **중립/대기** | `#3498DB` (Deep Blue) | 분석 중, 관리 가능 영역 (전략적 접근) | Loss/Gain 영역 사이의 배경, 측정값 표시 |

## 2. UI 컴포넌트 상태(State) 검증 및 미세 조정 상세 명세

### A. Loss Gauge (손실 게이지) 시각화 최종 정의

**목표:** 손실이 증가할수록 시각적 긴장감을 극대화하고, '생존 전략'에 대한 즉각적인 인식을 유도한다.

1.  **레이아웃:**
    *   **형태:** 반원(Arc) 또는 진행률 바(Progress Bar) 형태를 기본으로 하되, 위험도가 높을 경우 시각적 압박감을 주기 위해 **불규칙한 곡선(Distorted Arc)** 효과를 적용한다.
    *   **배경:** 전체 게이지 영역은 `Deep Blue` (`#004D66`)로 유지하여 전문성과 신뢰도를 확보한다.

2.  **색상 매핑 (Loss):**
    *   **Low Loss (안정):** `#3498DB` (Deep Blue) - 안정적 관리 영역.
    *   **Medium Loss (경고):** `Loss Gauge 채움 색상`을 점진적으로 **주황 계열**(`Orange`, `#F39C12`)로 전환 시작.
    *   **High Loss (위기/생존 위협):** `Loss Gauge 채움 색상`을 **강렬한 빨강**(`Red`, `#E74C3C`)으로 즉시 전환 및 경고 아이콘(🚨) 플래그 활성화.

3.  **텍스트 피드백 (Actionable Insight):**
    *   게이지 아래에 단순 수치 대신, **'경제적 생존 전략' 기반의 행동 지침**을 명확히 제시한다.
        *   *예시 (Loss가 높을 때):* "⚠️ 즉각적인 비용 절감 전략이 필요합니다. 다음 3단계 액션을 확인하세요."

### B. Red/Green 시스템 최종 정의

**목표:** 결과를 단순한 상태(State) 전달이 아닌, '전략적 방향성'으로 해석하게 한다.

1.  **Red State (위험):**
    *   **시각화:** `Loss Gauge`가 빨간색 영역에 진입하면, 전체 컴포넌트의 **주요 강조 색상**을 `#E74C3C`로 오버레이한다.
    *   **UI 요소:** 붉은색 배경 위에 **굵고 명확한 CTA 버튼(예: '긴급 조치 필요')**을 배치하여 즉각적인 행동을 요구한다.

2.  **Green State (성장):**
    *   **시각화:** `Gain Gauge`가 목표치를 초과 달성하면, 배경색을 `#2ECC71`로 변경하고 부드러운 애니메이션(Pulse)을 적용하여 성취감을 전달한다.
    *   **UI 요소:** 녹색 강조와 함께 **'성공 전략 발견 (Success Strategy Found)'**이라는 긍정적인 메시지를 중앙에 배치한다.

## 3. 최종 디자인 가이드라인 요약 (개발자 전달용)

| 요소 | 스펙/값 | 적용 방식 |
| :--- | :--- | :--- |
| **Primary Color** | Deep Blue (`#004D66`) | 기본 배경 및 신뢰성 유지 |
| **Risk Color** | Red (`#E74C3C`) | 손실 발생 시 경고/위기 표시 |
| **Success Color** | Growth Green (`#2ECC71`) | 긍정적 성장 및 전략 달성 표시 |
