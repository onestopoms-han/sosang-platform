### C. 개발팀 전달용 API 명세 (API Specification)

| API End Point | Method | Description |
| :--- | :--- | :--- |
| `/api/trust-widget` | GET | 신뢰도 점수 및 상태 (Default/High Risk) 조회 |
| `/api/pain-gauge` | GET | 불안 점수 및 상태 (Default/High Risk) 조회 |

**사용 예시:**
```javascript
// Trust Widget 데이터 조회
fetch('/api/trust-widget')
  .then(response => response.json())
  .then(data => {
    console.log(data); // { score: 85, status: 'default' }
  });

// Pain Gauge 데이터 조회
fetch('/api/pain-gauge')
  .then(response => response.json())
  .then(data => {
    console.log(data); // { score: 60, status: 'high_risk' }
  });
```
