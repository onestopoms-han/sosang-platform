# 📈 [Storyboarding Draft] 데이터 기반 성장 컨설팅 플랫폼 랜딩 페이지 초안
**(Target: 불안감 자극 → 해결책 제시 → 확신 부여 → 행동 유도)**

## 📄 전체 콘셉트 요약
*   **톤앤매너:** 신뢰감, 전문성(Authority), 명확함. (Deep Blue 계열의 지적이고 안정적인 느낌 강조)
*   **핵심 메시지:** "직감이나 경험에 의존하는 성장은 이제 그만. 데이터가 증명하는 성장 로드맵을 설계하세요."
*   **페이지 목표:** 방문자가 자신의 현재 상황(Pain Point)에 공감하고, 'ROI 시뮬레이션'이라는 구체적인 해결책의 가치를 느끼게 하여 Standard 플랜 가입 버튼을 누르게 한다.

***

## 🚀 섹션별 스토리보드 & 카피 초안 (총 6섹션 구조 권장)

### 🌟 Section 1: The Hook - 문제 제기 및 공감 유도 (Above the Fold)
**[목표]** 방문자의 현재 불안감을 즉각적으로 건드리며 시선을 붙잡는다.
**[비주얼 지시]** 소상공인이 머리를 감싸 쥐거나, 복잡한 그래프 앞에서 고민하는 실루엣 이미지 사용. 배경은 혼란스럽고 어두운 느낌에서 시작하여 플랫폼 로고가 비추는 곳부터 명확해지는 연출.

| 요소 | 카피 (Korean) | 역할/지시 사항 |
| :--- | :--- | :--- |
| **헤드라인 (H1)** | 🚨 아직도 ‘감’에 의존하여 가게를 운영하고 계신가요? | 질문형으로 시작하여 공감을 유도. '감'이라는 단어는 비전문적이고 불안정한 이미지를 부여함. |
| **서브 카피** | 소상공인의 노력만으로는 부족합니다. 2024년, 시장은 데이터로 움직입니다. 당신의 성장은 숫자로 증명되어야 합니다. | 현재의 어려움을 구조적인 문제(데이터 부재)로 정의하여 플랫폼이 필요함을 암시. |
| **핵심 CTA** | ✅ [무료 진단 받기] ROI 시뮬레이션으로 우리 가게 미래 예측하기 (Standard 플랜 강조 버튼) | 가장 강력한 기능(ROI 시뮬레이터)을 전면에 배치하고, '무료'로 심리적 장벽 제거. |

### 💡 Section 2: The Solution - 플랫폼 소개 및 가치 정의
**[목표]** 우리가 제공하는 것이 단순 컨설팅이 아닌 '시스템적인 해결책'임을 명확히 한다.
**[비주얼 지시]** 복잡했던 배경에서, 데이터가 깔끔하게 정리되어 성장 그래프를 만들어내는 애니메이션 연출.

| 요소 | 카피 (Korean) | 역할/지시 사항 |
| :--- | :--- | :--- |
| **제목 (H2)** | BDS는 소상공인에게 '직감'이 아닌 '데이터 기반의 성장 로드맵'을 제시합니다. | 핵심 가치('데이터 기반')를 다시 한번 강조하며 전문성을 각인시킨다. |
| **본문 카피** | 저희 플랫폼은 수많은 시장 변수와 소비 패턴 데이터를 AI로 분석하여, 사장님의 사업이 겪을 수 있는 모든 위험 요소를 미리 예측하고 최적의 성장 경로를 설계합니다. 단순히 '잘 될 것 같다'가 아니라, **‘몇 % 성장할 것이다’**라는 숫자로 보여드립니다. | 기능 나열 대신 '결과(숫자)'에 초점을 맞춘다. (Pain Point 해결에 집중) |

### 🔬 Section 3: The Core Feature - ROI 시뮬레이션 강조 (Proof & Mechanism)
**[목표]** 플랫폼의 존재 이유이자, 가장 매력적인 기능인 ROI 시뮬레이션을 깊이 있게 설명하여 구매 욕구를 폭발시킨다.
**[비주얼 지시]** 실제 계산기 인터페이스와 함께 'Before/After'를 대비시키는 대형 섹션 배치.

| 요소 | 카피 (Korean) | 역할/지시 사항 |
| :--- | :--- | :--- |
| **섹션 제목 (H2)** | 💰 [가장 강력한 무기] 막연함은 제거하고, 숫자로 미래를 디자인하세요: ROI 시뮬레이터 | '돈'과 관련된 단어(ROI)를 전면에 배치하여 경제적 이득에 집중하게 한다. |
| **서브 카피** | "혹시 마케팅 비용을 잘못 쓸까 봐 불안하다"라는 막연한 걱정은 이제 끝입니다. 저희 시뮬레이션기는 사장님의 초기 투자금 대비, 예상 수익성(ROI)을 단계별로 계산하여 가장 효율적인 성장 플랜을 즉시 알려드립니다. | '불안감'이라는 Pain Point를 직접 언급하며 해결책으로 제시. |
| **강조 포인트 (3가지)** | 1. **예측 정확도:** 국내외 최신 데이터를 실시간 반영합니다. / 2. **단계별 로드맵:** A 단계에서 B 단계로 가기 위한 구체적 액션 플랜을 제공합니다. / 3. **직관성:** 복잡한 그래프 대신, 이해하기 쉬운 '만족도/수익률' 게이지로 보여줍니다. | 기능의 우월성을 객관적인 수치와 프로세스로 증명한다. |

### 🏷️ Section 4: The Value Comparison - 플랜 비교 (Standard vs Basic)
**[목표]** Standard 플랜을 '필수적이고 완벽한 선택지'로 보이게 하고, Basic은 '미완의 경험'처럼 느끼게 한다.
**[비주얼 지시]** 3단 구조의 표(Table) 형태를 유지하되, **Standard 열 전체에 Growth Green 계열 하이라이트 처리.**

| 플랜 | Basic (₩19,900) | ⭐ Standard (₩49,900) - 권장 |
| :--- | :--- | :--- |
| **핵심 메시지** | ✨ 성장 방향을 '엿볼 수 있는' 맛보기 경험. | 🚀 데이터로 성장의 전 과정을 '완벽하게 설계하는' 완성형 솔루션. |
| **주요 기능** | [기본 컨설팅 리포트] 제공 (텍스트 위주) | **[실시간 ROI 시뮬레이터]** + 맞춤 전략 로드맵 (숫자로 증명) |
| **가치 비교 카피** | 💡 '어디서부터' 시작해야 할지 감을 잡아줍니다. | 📈 '무엇을', '얼마나' 투자해야 할지 명확한 답을 얻습니다. |

### ✅ Section 5: Guarantee & Trust - 신뢰도 구축
**[목표]** 구매 결정의 마지막 망설임을 해소하고 안전함을 제공한다.
**[비주얼 지시]** 고객사 로고(가상의 성공 사례) 배치, 또는 데이터 분석가가 진지하게 설명하는 이미지 사용.

| 요소 | 카피 (Korean) | 역할/지시 사항 |
| :--- | :--- | :--- |
| **제목 (H2)** | BDS와 함께 성장한 소상공인들의 실제 변화 기록 | 성공 사례(Testimonial)를 통해 사회적 증거(Social Proof) 제공. |
| **보증 문구** | "우리 가게의 데이터는 비밀입니다. 오직 사장님의 성장을 위한 정보로만 활용합니다." (개인정보 보호 강조) | 소상공인이 가장 우려하는 '데이터 유출' 리스크를 선제적으로 차단하여 신뢰도 확보. |

### 🛒 Section 6: Final CTA - 행동 촉구
**[목표]** 망설임을 즉시 '행동'으로 전환시킨다.
**[비주얼 지시]** 가장 크고 눈에 띄는 버튼과 함께, "지금 바로 시작하세요!"라는 긴급성을 부여하는 타이머/배너 사용.

| 요소 | 카피 (Korean) | 역할/지시 사항 |
| :--- | :--- | :--- |

| **마지막 CTA** | 🥇 [Standard 플랜으로 성공 로드맵 완성하기] / 🥈 [Basic 플랜으로 먼저 진단받기] | Standard를 가장 크고 매력적인 색상으로 강조하며, 구매 결정을 최종적으로 유도한다. |
