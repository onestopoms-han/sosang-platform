# 💼 현빈 — MVP 기능 우선순위 및 초기 공급자 Pilot 모집 전략

## 1. MVP 기능 우선순위 재조정 (MVP Phase 1)

**목표**: 가장 빠른 피드백을 받을 수 있는 핵심 기능을 집중 개발하여, 가격 전략과 사용자 여정의 흐름이 실제로 작동하는지 검증합니다.

| 순위 | 기능 | 기술적 구현 요구사항 | 비즈니스 가치 |
| :--- | :--- | :--- | :--- |
| **P0 (Must)** | **공급자 등록 & 상품 리스팅** | - 공급자용 웹폼 (반드시 인증서/사업자번호 입력)<br>- 카테고리별 자동 태그 및 가격 비교 기능<br>- 이미지 업로드 (최소 1~3 장) | - 초기 상품 풀 형성의 핵심<br>- 공급자의 진입 장벽 낮춤 |
| **P0 (Must)** | **신뢰도 위젯 (Trust Widget)** | - API 연동 후 실시간 신뢰 점수 표시<br>- 상태별 색상 (`Deep Blue`/`Growth Green`/`Red`) 적용 | - 구매자 신뢰도 확보<br>- 가격 전략의 신뢰성 증명 |
| **P1 (Should)** | **결제 시스템 기본 구현** | - 간편 결제 (네이버페이, 카카오페이)<br>- 자동 이체 및 포인트 적립 로직 | - 즉각적인 매출 전환<br>- 가격 책정의 현실적 검증 |
| **P2 (Could)** | **AI 기반 추천 상품** | - 사용자 여정 단계별 추천 알고리즘<br>- 초기에는 규칙 기반 (규칙형 추천) | - 전환율 극대화<br>- 사용자 경험 개선 |

**우선순위 재조정 근거**:  
- **가격 전략**: Pro Plan (3~5 만원대) 이 가장 수익성 높은 구간이므로, P0 기능으로 이를 지지할 수 있는 상품 리스팅과 결제 기능을 우선시합니다.
- **사용자 여정**: 초기 사용자가 '신뢰도'를 가장 중요하게 여기므로, Trust Widget 을 즉시 노출하는 것이 전환율을 높입니다.

## 2. 초기 공급자 Pilot 그룹 모집 전략 (Pilot Program v1)

**목표**: MVP 검증에 가장 적합하고 가격 전략을 지지할 수 있는 공급자 5~7 곳을 확보합니다.

### 🎯 타겟 페르소나 및 인센티브

| 공급자 유형 | 타겟 상품 예시 | 모집 인센티브 | 기대 효과 |
| :--- | :--- | :--- | :--- |
| **대중형 (Mass)** | 일반 식료품, 음료수, 일상용품 | - 1 개월 무료 수수료<br>- 마케팅 포인트 20% 할증 | - 초기 트래픽 확보<br>- 기본 가격대 검증 |
| **프리미엄 (Premium)** | 유기농 식품, 한정판 커피, 고급 와인 | - 전용 브랜드 스토어 제공<br>- AI 추천 알고리즘 우선 적용 | - 고가치 상품 풀 형성<br>- Pro Plan 고객 유도 |
| **특수형 (Specialty)** | 지역 특산품, 계절식, DIY 키트 | - SNS 홍보 지원 (레오/YouTube)<br>- 맞춤형 제품 페이지 디자인 | - 차별화된 스토리텔링<br>- 타겟 페르소나 확립 |

- **신뢰도 점수**: 평균 4.5 점 이상
