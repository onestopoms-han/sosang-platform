# BDS 소상공인플랫폼 — 가치 검증 인터뷰 스크립트 (초안)

## 🎯 목적 및 컨셉

- **주제:** 플랫폼이 해결해야 할 핵심 문제와 실제 소상공인의 '숫자'를 통한 고민 파악.
- **대상:** 플랫폼을 고려하거나 현재 어려움을 겪고 있는 소상공인, 창업 준비생.
- **분위기:** 전문적인 컨설턴트 인터뷰 (신뢰감 + 공감).

---

## 1️⃣ 도입부: 신뢰 형성 & 상황 설정 (5 분)

| 구분 | 내용/대본 초안 | 비고 |
|------|------------------|-------|
| **[개막]** | "안녕하세요, [이름]님. 오늘 시간 내주셔서 감사합니다.<br>저는 BDS 소상공인플랫폼 팀에서 일하는 [직함]입니다.<br>오늘은 [이름]님의 사업 현황과 고민을 조금 더 깊게 들어보고,<br>혹시 우리 플랫폼이 도움이 될 수 있는 부분이 있는지 함께 나누고 싶습니다."<br><br>**[목표]**<br>- 비즈니스 파트너로서 전문성을 보여주기<br>- 인터뷰에 대한 거부감을 줄이고 편안하게 답변 유도 | |
| **[현재 상황 확인]** | "먼저 [이름]님의 현재 사업과 운영 상태를 간단히 파악해 보겠습니다.<br>1. 어떤 업종인가요?<br>2. 몇 년 차 사업주신가요?<br>3. 주로 어디에 고객님을 모시나요?"<br><br>**[목표]**<br>- 인터뷰의 맥락을 설정하고, 이후 질문과의 연결 고리 마련 | |

---

## 2️⃣ 본론: KPI & Pain Point 중심 심층 질문 (15 분)

### 📊 AOV(평균 거래액) 관점 — "매출을 늘리는 데 어떤 고민이신가요?"

| 구분 | 내용/대본 초안 | 비고 |
|------|------------------|-------|
| **[현황]** | "지금 매출을 보실 때 가장 먼저 떠오르는 숫자는?<br>예: '월간 매출', '평균 한 번 주문당 금액' 등"<br><br>**[목표]**<br>- 현재 AOV 수준을 파악하고, 플랫폼의 'AOV 증대 솔루션' 필요성 연결 | |
| **[고민]** | "매출을 늘리려고 노력하신다면, 어떤 방법을 주로 사용하시나요?<br>(할인, 이벤트, 홍보물 배포 등)<br>그런데 그 방법이 효과는 있지만 비용은 많이 들어가요. 혹시 '비용 대비 매출 증가'가 고민이시나요?"<br><br>**[목표]**<br>- 현재 마케팅 비용 구조 파악<br>- AOV 증대 솔루션 필요성 암시 | |
| **[도전]** | "혹시 '단품 판매로 매출을 높이는 것보다', '다품 판매나 업사이클링'으로 평균 거래액을 늘릴 수 있다면 좋겠다'는 생각을 해보셨나요?<br>만약 그런 방법을 추천해 주신다면, 어떤 제품/서비스를 먼저 팔고 싶으신가요?"<br><br>**[목표]**<br>- 다품 판매 전략에 대한 잠재적 니즈 파악<br>- 플랫폼의 '상품 추천/업사이클링' 솔루션 연결 | |

### 💸 CAC(고객 획득 비용) 관점 — "고객을 유지하고 새로 모시는 데 어떤 어려움이 있나요?"

| 구분 | 내용/대본 초안 | 비고 |
|------|------------------|-------|
| **[비용]** | "현재 마케팅 비용 중 가장 큰 비중을 차지하는 항목은?<br>(인스타그램 광고, 블로그 포스팅, 지역 홍보물 등)<br>그 비용을 통해 얼마나 많은 고객님을 모셨나요?"<br><br>**[목표]**<br>- CAC 수준 파악<br>- 현재 채널별 효율성 비교 | |
| **[효과]** | "고객을 모신 후, 다시 방문하거나 재구매를 유도하는 데 어떤 어려움을 느끼시나요?<br>예: '재방문율이 낮다', '할인을 안 해도 안 온다' 등<br><br>**[목표]**<br>- 고객 유지 비용 (Retention Cost) 문제 인식<br>- LTV(생애 가치) 증대 전략 필요성 암시 | |
| **[도전]** | "고객을 유지하기 위해 가장 노력하신 점은?<br>(할인, 포인트, VIP 멤버십 등)<br>만약 '고객이 재방문하고 싶게 만드는 것'에 투자한다면, 어떤 서비스를 선호하시나요?"<br><br>**[목표]**<br>- 고객 유지 솔루션 니즈 파악<br>- 플랫폼의 '멤버십/리워드' 시스템 연결 | |

### 🔄 LTV(고객 생애 가치) 관점 — "고객과 관계를 만들어가는 데 어떤 고민이신가요?"

| 구분 | 내용/대본 초안 | 비고 |
|------|------------------|-------|
| **[관계]** | "고객님과 관계를 이어가려면, 어떤 방법이 가장 좋다고 생각하나요?<br>예: '정기적인 이벤트', '개인화된 추천' 등<br><br>**[목표]**<br>- 고객 관계 관리 (CRM) 니즈 파악 | |
| **[도전]** | "고객을 '단순한 구매자'가 아닌, '브랜드 애호가'로 만들어주려면, 어떤 것이 필요하다고 느끼시나요?<br>예: '제품 리뷰 공유', '특정 제품 사용법 교육' 등<br><br>**[목표]**<br>- 브랜드 충성도 (Brand Loyalty) 증대 전략 니즈 파악 | |
| **[가치]** | "고객 생애 가치를 높이는 데, 가장 큰 장벽은 무엇이라고 생각하나요?<br>예: '비용', '시간', '기술적 어려움' 등<br><br>**[목표]**<br>- LTV 증대 솔루션 필요성 명확화 | |

---

## 3️⃣ 플랫폼 가치 제안 및 다음 단계 (5 분)

| 구분 | 내용/대본 초안 | 비고 |
|------|------------------|-------|
| **[플랫폼 소개]** | "지금까지 [이름]님의 고민을 잘 들었습니다.<br>우리의 BDS 소상공인플랫폼은 바로 이런 'AOV', 'CAC', 'LTV' 문제를 해결하기 위해 기획된 플랫폼입니다.<br>특히, '다품 판매 전략'이나 '고객 유지 솔루션' 등을 통해 매출과 수익을 동시에 높일 수 있도록 지원해 줍니다."<br><br>**[목표]**<br>- 플랫폼의 핵심 가치 명확히 전달<br>- KPI Pain Point 해결책 제시 | |
| **[다음 단계 제안]** | "오늘 인터뷰로 얻은 정보를 바탕으로,<br>우리 플랫폼이 [이름]님에게 어떤 도움을 줄 수 있을지 구체적인 시나리오를 만들어 드리겠습니다.<br>혹시 지금 바로 시연 (Demo) 을 보여드리면 좋을까요?<br><br>**[목표]**<br>- 다음 단계(데모/상담)로 자연스럽게 연결<br>- 인터뷰 종료 후 추가 행동 유도 | |

---

## 📝 인터뷰 전 체크리스트 (인터뷰어용)

- [ ]受访者의 업종/차수/현재 채널 파악
- [ ] AOV, CAC, LTV 관련 질문이 자연스럽게 이어지도록 흐름 설계
- [ ] 응답자가 불편해하면 '예전 경험'이나 '비즈니스 뉴스' 등으로 전환
- [ ] 인터뷰 종료 후 바로 피드백 수집 (녹취/사진 등)

---

## 💡 참고 자료

- **KPI Pain Point 연결:** 
  - AOV → 다품 판매, 업사이클링 솔루션
  - CAC → 고객 유지 비용 최적화 전략
  - LTV → 브랜드 충성도 및 고객 관계 관리 (CRM)
- **플랫폼 가치 제안:** 'AOV/CAC/LTV' 문제를 해결하는 솔루션 구체적 연결
