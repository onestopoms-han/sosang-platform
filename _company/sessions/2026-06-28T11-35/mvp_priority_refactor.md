## ✅ MVP 핵심 기능 3 가지 (API 불확실성 대응 버전)

### 1️⃣ **송이버섯 판매 플랫폼 — 기본 거래 시스템**  
- **기능**: 소상공인(판매자) ↔ 소비자(구매자) 간 송이버섯 거래 매칭
- **MVP 필수**: 상품 등록, 주문 처리, 결제 연동 (PayPal 연동 시도 중)
- **우선순위**: 🔴 최상 (기반 인프라)

### 2️⃣ **Trust Widget 및 PainGauge — 신뢰·성장 시각화**  
- **기능**: 판매자 신뢰도 점수 (Deep Blue), 성장 지표 (Green/Yellow/Red)
- **MVP 필수**: API 없이도 HTML/JS 로 구현 가능 (코다리가 코드 작성 중)
- **우선순위**: 🟡 2 위 (신뢰 메커니즘)

### 3️⃣ **Pilot 데이터 수집 프로세스 — 최소 기술 요구**  
- **기능**: 판매자 기본 정보, 초기 송이버섯 재고, 고객 피드백
- **MVP 필수**: 구글 폼/엑셀 또는 간단한 API 로 대체 가능 (코다리가 파일 접근 실패 시)
- **우선순위**: 🟢 3 위 (데이터 기반 개선)
