# 🎬 사용자 온보딩 스토리라인 릴스 기획안 (Beta Tester Recruitment Campaign)

## 📌 개요
- **목표**: MVP 출시 2 주 전부터 베타 테스트 소상공인 유치 및 플랫폼 가치 전달  
- **형식**: 60 초 릴스 + 피드 영상 (Instagram Reels, Feed)  
- **출시 일정**: MVP 출시일 기준 T-14 일부터 시작

## 🎯 핵심 메시지
> "당신의 가게를 위한 AI 파트너, 옆집선배가 찾아왔습니다. 데이터 기반 컨설팅으로 매출 30% 성장하세요."

## 📖 스크립트 (60 초)

| 시간 | 화면 | 음성/오버레이 | 비고 |
|------|------|----------------|-------|
| **0-5 초** |忧心表情 소상공인 가게 전경 → 클로즈업 | "매출이 늘지 않는 이유, 알고 계신가요?" | 훅: 문제 제기 |
| **5-15 초** | PainGauge 결과 표시 (예시: '고객 유입 부족') + 옆집선배 등장 | "가게의 숨은 문제를 AI 가 찾아냅니다." | 캐릭터 애니메이션 시작 |
| **15-30 초** | 플랫폼 UI 모션 그래픽 (온보딩, 데이터 분석 섹션) + 성장 그린 컬러 강조 | "BDS 소상공인플렛폼 이 당신을 도와드립니다." | 기능 데모 |
| **30-45 초** | Trust Widget 예시 (실제 성과 데이터 가상) → 옆집선배가 설명하는 듯한 손짓 | "신뢰할 수 있는 컨설팅, 바로 지금 시작하세요." | 신뢰 구축 |
| **45-55 초** | '베타 테스트 참여 신청' 버튼이 화면에 나타남 → 옆집선배 인형 등장 | "무료 컨설팅 포함! 신청 방법은 댓글에서 확인하세요." | 행동 유도 (CTA) |
| **55-60 초** | 브랜드 로고 + 해시태그 오버레이 | "당신의 가게를 위한 AI 파트너, 옆집선배!" | 마무리 |
