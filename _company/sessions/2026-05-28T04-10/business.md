# 📊 핵심 지표 (KPI) - 소상공인의 어려움 해결 우선순위 매핑 문서 (Version 1.0)

**작성일:** 2026-05-31  
**작성자:** 💼 현빈 (비즈니스 전략가)  
**수신:** 코다리(Developer), Designer, 레오(Writer)  

---

## 🎯 목적
PM 대시보드의 모든 핵심 지표(R01, S01 등) 구현 시 '소상공인의 실질적 어려움 해결'이라는 BDS의 본질적 가치를 반드시 반영해야 합니다. 이 문서는 각 KPI가 어떤 Pain Point 를 해결하는지 명시하고, 개발팀이 로직 구현 시 목표 달성 기준을 재검토할 수 있도록 우선순위를 제공합니다.

---

## 📌 핵심 지표 (KPI) 정의 및 해결 어려움 매핑

### R01: Revenue Health Index (수익 건강 지수)
- **해결 어려움:** 매출 감소, 현금 흐름 불안정  
- **목표 KPI 달성 로직:**  
  - 임계값 설정: 월 매출 대비 목표 수익률 (예: 15%) 이하 → 경고 발령  
  - 데이터 연동: 실제 PayPal 거래 데이터 (실패 시 placeholder 로 대체 불가)  
  - 우선순위: 🔴 최고  
- **구현 시 참고:**  
  - 단순 매출 집계 X, '실질적 수익' 개념으로 접근 (수수료 제외, 세금 포함)  
  - 목표: 소상공인이 KPI 를 보고 "이번 달 생존에 위협이 된다"고 느낄 수 있도록 직관적인 시각화

### S01: Supply Chain Stability Score (공급망 안정성 점수)
- **해결 어려움:** 원자재 가격 상승, 배송 지연, 재고 부족  
- **목표 KPI 달성 로직:**  
  - 공급원별 평균 납기일 대비 실제 납품률 계산  
  - 임계값 설정: 공급망 지연으로 인한 매출 손실 > 월 수익의 10% → 대안 제시  
  - 데이터 연동: 실시간 원자재 가격 (외부 API 또는 수동 입력)  
  - 우선순위: 🟠 중상  
- **구현 시 참고:**  
  - 단순 납품률 계산 X, '경제적 손실'로 변환하여 시각화  
  - 목표: "이 공급원은 위험이다"라고 바로 판단 가능

### M01: Marketing Efficiency Ratio (마케팅 효율 비율)
- **해결 어려움:** 마케팅 예산 낭비, 신규 고객 확보 비용 높음  
- **목표 KPI 달성 로직:**  
  - ROI = (신규 매출 - 광고비) / 광고비 × 100%  
  - 임계값 설정: ROI < 50% → 예산 재배분 권장  
  - 데이터 연동: 실제 광고 클릭, 전환 데이터 (유튜브, 인스타그램 등)  
  - 우선순위: 🟢 중하  
- **구현 시 참고:**  
  - 단순 클릭 수 X, '실질적 고객 확보'와 연결  
  - 목표: "이 채널은 효율적이다"라고 바로 판단

### C01: Customer Satisfaction Score (고객 만족도 점수)
- **해결 어려움:** 재구매율 낮음, 부정적 리뷰 증가  
- **목표 KPI 달성 로직:**  
  - NPS = 추천 고객 비율 - 비추천 고객 비율  
  - 임계값 설정: NPS < 30 → 서비스 개선 권장  
  - 데이터 연동: 실제 고객 평가 (유튜브 댓글, 인스타그램 반응 등)  
  - 우선순위: 🟢 중하  
- **구현 시 참고:**  
  - 단순 평점 X, '실질적 불만'과 연결  
  - 목표: "이 고객이 다시 오지 않을 수 있다"라고 바로 판단

  - KPI 가 '감정적 만족' (고객 신뢰 향상) 을 가져오는지? → C01 은 ⚠️ (부수적 효과)  
- **데이터 모델 우선순위:**  
  - 단순 데이터 집계 X, '해결책 제시' 로직을 포함해야 함  
  - 예: R01 임계값 도달 시 "이러한 경우, 다음 행동을 권장합니다" 메시지 자동 표시
