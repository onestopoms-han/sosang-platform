## 2. 핵심 진단 지표 (Key Diagnostic Indicators)

### **KPI_01: 재정 건강도 (Financial Health Score)**
- **목표:** 소상공인의 즉각적인 자금 부족 위험을 예측.
- **정의:** `(현금 보유 + 예상 매입 자금) / (월 고정 비용 * 6 개월)`를 기반으로 한 점수.
- **계산 로직:**
    - `numerator = current_balance + expected_inflow`
    - `denominator = fixed_monthly_cost * 6`
    - `score = numerator / denominator`
- **위험도 매핑:**
    - `score >= 1.5`: Normal (정상)
    - `0.8 <= score < 1.5`: Warning (주의)
    - `score < 0.8`: Critical (위험)
- **데이터 신뢰도 확보 방법:** 실제 은행 거래 내역 API 연동 시나리오, 역사적 데이터 분석.

### **KPI_02: 공급망 안정성 (Supply Chain Stability Index)**
- **목표:** 원자재 가격 변동 및 납기 지연 가능성을 예측.
- **정의:** `(지난 6 개월 평균 원가 상승률) + (평균 배송 지연 일수 * 가중치)`를 기반으로 한 점수.
- **계산 로직:**
    - `cost_inflation = sum(average_monthly_price_increase_6mo)`
    - `delivery_delay_weighted = average_delivery_delay_days * 0.5`
    - `score = cost_inflation + delivery_delay_weighted`
- **위험도 매핑:**
    - `score <= 2.0`: Normal (정상)
    - `2.1 <= score < 4.0`: Warning (주의)
    - `score >= 4.0`: Critical (위험)
- **데이터 신뢰도 확보 방법:** 외부 경제 데이터 API 연동, 내부 재고 데이터 분석.

### **KPI_03: 디지털 전환 잠재력 (Digital Adoption Potential)**
- **목표:** AI 도구 도입 효과를 예측하고 적합한 솔루션을 추천.
- **정의:** `온라인 매출 비중 + 기존 도구의 활용도`를 기반으로 한 점수.
- **계산 로직:**
    - `online_sales_ratio = online_revenue / total_revenue`
    - `existing_tool_usage = (used_tools_count / total_available_tools)`
    - `score = online_sales_ratio * 0.5 + existing_tool_usage * 0.5`
- **위험도 매핑:**
    - `score >= 0.6`: Normal (높은 잠재력)
    - `0.3 <= score < 0.6`: Warning (보통)
    - `score < 0.3`: Critical (낮은 잠재력)
- **데이터 신뢰도 확보 방법:** 플랫폼 내 성과 데이터 분석, 외부 시장 트렌드 DB 연동.

## 2. 핵심 진단 지표 (Key Diagnostic Indicators)

### **KPI_01: 재정 건강도 (Financial Health Score)**
- **목표:** 소상공인의 즉각적인 자금 부족 위험을 예측.
- **정의:** `(현금 보유 + 예상 매입 자금) / (월 고정 비용 * 6 개월)`를 기반으로 한 점수.
- **계산 로직:**
    - `numerator = current_balance + expected_inflow`
    - `denominator = fixed_monthly_cost * 6`
    - `score = numerator / denominator`
- **위험도 매핑:**
    - `score >= 1.5`: Normal (정상)
    - `0.8 <= score < 1.5`: Warning (주의)
    - `score < 0.8`: Critical (위험)
- **데이터 신뢰도 확보 방법:** 실제 은행 거래 내역 API 연동 시나리오, 역사적 데이터 분석.

### **KPI_02: 공급망 안정성 (Supply Chain Stability Index)**
- **목표:** 원자재 가격 변동 및 납기 지연 가능성을 예측.
- **정의:** `(지난 6 개월 평균 원가 상승률) + (평균 배송 지연 일수 * 가중치)`를 기반으로 한 점수.
- **계산 로직:**
    - `cost_inflation = sum(average_monthly_price_increase_6mo)`
    - `delivery_delay_weighted = average_delivery_delay_days * 0.5`
    - `score = cost_inflation + delivery_delay_weighted`
- **위험도 매핑:**
    - `score <= 2.0`: Normal (정상)
    - `2.1 <= score < 4.0`: Warning (주의)
    - `score >= 4.0`: Critical (위험)
- **데이터 신뢰도 확보 방법:** 외부 경제 데이터 API 연동, 내부 재고 데이터 분석.

### **KPI_03: 디지털 전환 잠재력 (Digital Adoption Potential)**
- **목표:** AI 도구 도입 효과를 예측하고 적합한 솔루션을 추천.

- **계산 로직:**
    - `online_sales_ratio = online_revenue / total_revenue`
    - `existing_tool_usage = (used_tools_count / total_available_tools)`
    - `score = online_sales_ratio * 0.5 + existing_tool_usage * 0.5`
- **위험도 매핑:**
    - `score >= 0.6`: Normal (높은 잠재력)
    - `0.3 <= score < 0.6`: Warning (보통)
    - `score < 0.3`: Critical (낮은 잠재력)
- **데이터 신뢰도 확보 방법:** 플랫폼 내 성과 데이터 분석, 외부 시장 트렌드 DB 연동.

*   **기술적 신뢰도 확보 포인트:** '디지털 전환 잠재력 (Digital Adoption Potential)'을 측정하여, 어떤 도구가 적합한지 추천하고 유동성을 지원해야 함.

## 2. 핵심 진단 지표 (Key Diagnostic Indicators)

### **KPI_01: 재정 건강도 (Financial Health Score)**
- **목표:** 소상공인의 즉각적인 자금 부족 위험을 예측.
- **정의:** `(현금 보유 + 예상 매입 자금) / (월 고정 비용 * 6 개월)`를 기반으로 한 점수.
- **계산 로직:**
    - `numerator = current_balance + expected_inflow`
    - `denominator = fixed_monthly_cost * 6`
    - `score = numerator / denominator`
- **위험도 매핑:**
    - `score >= 1.5`: Normal (정상)
    - `0.8 <= score < 1.5`: Warning (주의)
    - `score < 0.8`: Critical (위험)
- **데이터 신뢰도 확보 방법:** 실제 은행 거래 내역 API 연동 시나리오, 역사적 데이터 분석.

### **KPI_02: 공급망 안정성 (Supply Chain Stability Index)**
- **목표:** 원자재 가격 변동 및 납기 지연 가능성을 예측.
- **정의:** `(지난 6 개월 평균 원가 상승률) + (평균 배송 지연 일수 * 가중치)`를 기반으로 한 점수.
- **계산 로직:**
    - `cost_inflation = sum(average_monthly_price_increase_6mo)`
    - `delivery_delay_weighted = average_delivery_delay_days * 0.5`
    - `score = cost_inflation + delivery_delay_weighted`
- **위험도 매핑:**
    - `score <= 2.0`: Normal (정상)
    - `2.1 <= score < 4.0`: Warning (주의)
    - `score >= 4.0`: Critical (위험)
- **데이터 신뢰도 확보 방법:** 외부 경제 데이터 API 연동, 내부 재고 데이터 분석.

### **KPI_03: 디지털 전환 잠재력 (Digital Adoption Potential)**
- **목표:** AI 도구 도입 효과를 예측하고 적합한 솔루션을 추천.
- **정의:** `온라인 매출 비중 + 기존 도구의 활용도`를 기반으로 한 점수.
- **계산 로직:**
    - `online_sales_ratio = online_revenue / total_revenue`
    - `existing_tool_usage = (used_tools_count / total_available_tools)`
    - `score = online_sales_ratio * 0.5 + existing_tool_usage * 0.5`
- **위험도 매핑:**
    - `score >= 0.6`: Normal (높은 잠재력)
    - `0.3 <= score < 0.6`: Warning (보통)
    - `score < 0.3`: Critical (낮은 잠재력)
- **데이터 신뢰도 확보 방법:** 플랫폼 내 성과 데이터 분석, 외부 시장 트렌드 DB 연동.
