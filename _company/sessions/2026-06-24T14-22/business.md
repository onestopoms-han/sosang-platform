# BDS 플랫폼 베타 수익화 및 KPI 연계형 데이터 기반 진단 로드맵 (Structure Draft)

## 1. 개요 (Executive Summary)
**목표**: 소상공인플렛폼의 베타 테스트를 성공적으로 진행하면서 초기 트래픽 확보와 동시 수익 창출 모델을 검증하는 것.  
**핵심 전략**: "기술적 신뢰도"를 기반으로 한 **데이터 기반 진단**을 진입 장벽이자 마케팅 자산으로 활용하고, 이를 단계적인 **수익화 (Monetization)** 로 연결하여 LTV(생애 가치) 를 극대화함.

---

## 2. 핵심 가치 제안 및 수익 모델 구조 (Freemium Model)

### 2-1. 제품 라이어 (Product Layers)
| 레이어 | 기능 명세 | 가격 정책 | 타겟 고객 | 수익화 포인트 |
| :--- | :--- | :--- | :--- | :--- |
| **L1: 무료 진단** | 위험도 매핑 기준 기반 핵심 지표 5 개 요약 (재정, 공급망 등) | Free | 모든 소상공인 | 트래픽 확보, 신뢰도 형성 |
| **L2: 유료 보고서** | L1 데이터 + AI 분석 리포트 (위험 원인 및 대응책 포함) | $9.99/회 또는 $49/월 | 문제 해결을 원하는 주류 사용자 | 단회 판매, 재구매 유도 |
| **L3: AI 컨설팅 구독** | 실시간 경고 + 맞춤형 액션플랜 생성 + 전문가 연결 중개 | $29.99/월 (초기 50% 할인) | 성장하려는 진지한 사업자 | 지속적 수익, 네트워크 효과 |
| **L4: B2B 파트너십** | L3 구독자 기반 광고/마케팅 솔루션 배급 수수료 | CPA/CPC | 플랫폼 운영사 및 외부 파트너 | 거대 비즈니스 확장성 확보 |

### 2-2. 가격 전략 (Pricing Strategy)
- **초기 베타 기간**: L1 무료, L2 $9.99, L3 $24.99 (입문자 우대가 적용)
- **가격 심리**: "무료"로 접근시킨 뒤, 구체적인 해결책은 유료 버전에서 제시하는 방식 채택.
- **라이선스 모델**: L3 구독자의 5% 를 외부 마케팅 파트너 (인플루언서 등) 에 배당하여 참여 유도.

## 3. KPI 측정 체계 및 연동 매트릭스 (KPI Matrix)

각 수익화 단계별 전환율을 추적하는 핵심 지표와 목표치를 정의함.

| 단계 | 주요 KPI (Leading/Lagging) | 목표치 (베타 기간 3 개월) | 연동 데이터 소스 |
| :--- | :--- | :--- | :--- |
| **유입** | - 유입 트래픽<br>- L1 무료 진단 참여율 | L1: 일 평균 500 명<br>참여율: 60% 이상 | `DiagnosisOutputSchema v2` API 호출 횟수, 웹 로그 |
| **전환 (L1 → L2)** | - 보고서 구매 전환율<br>- 재방문 주기 | 전환율: 8~12%<br>재방문: 주 1 회 평균 30% 상승 | 결제 시스템 데이터, 사용자 행동 로그 |
| **수익화 (L2 → L3)** | - 구독 가입 전환율<br>- AOV (평균 주문 금액) | 전환율: 5~7%<br>AOV: $35 이상 | 구독 결제 모듈, 재구매 기록 |
| **성숙** | - NPS (추천 의지)<br>- Churn Rate (탈퇴율) | NPS > 40<br>Churn < 5%/월 | 사용자 피드백, 사용 빈도 데이터 |

### 3-1. KPI 연동 로직 예시
```json
// API 응답 스키마에 포함될 예측 지표 구조 (예상치)
{
    "predicted_revenue": {
        "l2_report_sales": "$X,XXX", 
        "l3_subscription_retention": "XX%"
    },
    "kpi_correlation": {
        "risk_score_high": 0.85,   // 위험도 점수가 높을수록 L3 구독 전환율이 X% 상승
        "supply_chain_alert_count": 2.1 // 공급망 경고 횟수가 많을수록 보고서 구매 확률 Y% 증가
    }
}
```

## 4. 베타 참여자 유도 및 전환 흐름 (GTM Flow)

### 4-1. 마케팅 채널별 메시지 전략
| 채널 | 핵심 메시지 | 수익화 연결점 |
| :--- | :--- | :--- |
| **유튜브** | "소상공인, 당신의 가게 위험도를 AI 가 분석했습니다." → 보고서 미리보기 링크 (L2 구매 유도) | 영상 내 링크 클릭 추적 |
| **Instagram** | "오늘의 위험 체크: 공급망 중단 위험 80%?" → 상세 분석 페이지 (L3 구독 전환) | 릴스/이미지 카피에 CTA 포함 |
| **공식 홈페이지** | "무료 진단부터 시작하세요" → L1 진입 → L2 추천 | 랜딩 페이지 내 가이드 투어 |

### 4-2. 데이터 기반 참여자 모집 전략
- **타겟 페르소나**: 현재 1 인 기업, 전통시장 가맹점주, 온라인 판매 전환을 고려하는 소상공인.
- **참여 유도 Hook**: "무료 진단으로 위험도를 파악하세요" → L1 사용 후 "해결책 보고서는 $9.99 로 확인하세요".
- **베타 참여자 혜택**: 초기 1000 명에게 L3 구독 3 개월 무료 제공 (충성도 확보 및 단가 테스트).

## 5. 예상 ROI 및 수익성 분석 (Financial Projection)

### 5-1. 단위 경제학 (Unit Economics)
| 항목 | 예상치 | 비고 |
| :--- | :--- | :--- |
| 고객 획득 비용 (CAC) | $20 | 유튜브/인스타 광고 + 제휴 마케팅 포함 |
| 평생 가치 (LTV) | $85 | L2 구매 $10 x 3 회 + L3 구독 $30 x 9 개월 = $270 - 유지비용 |
| 마진율 | 65% | 기술 기반 비용 효율성 |

### 5-2. 베타 기간 목표 수익
- **트래픽**: 월 1 만 명 → L1 참여 6 천 명 → L2 구매 700 명 ($7,000) + L3 구독 100 명 ($3,000) = **약 $10,000 / 월**
- **성장 목표**: 3 개월 내 CAC 감소 20%, LTV 증가 40% 달성.
