## ✅ [최종 보고서] 지역 생산자 데이터 수집 및 표준화 프로세스

### 📌 I. 프로젝트 목표 요약 (Mission Statement)
*   **목표:** 단순 상품 정보(SKU) 확보를 넘어, 생산 과정과 스토리를 데이터로 구조화하여 플랫폼의 **객단가 상승(AOV 증진)** 동력으로 활용한다.
*   **핵심 측정 지표:** 스토리텔링 모듈 기여도 (정량적/정성적), 공급망 투명성 확보 정도.

### 📌 II. To-do List: 단계별 실행 가이드라인 (Action Plan)

| 단계 | 주체 및 역할 | 활동 내용 (To-do Item) | 기대 산출물 |
| :--- | :--- | :--- | :--- |
| **STEP 1** <br/>(준비 및 교육) | 영숙/전담 매니저 | **[데이터 표준화 워크숍 진행]**: 지역 생산자 대상, 데이터 입력의 중요성 및 포맷 가이드라인 설명. <br/>- *핵심:* '왜 이 데이터를 모으는지' (판매와 연결됨)에 대한 명확한 교육 필요. | 교육 자료(PPT), 참여 생산자 명단, 현장 질의응답 기록. |
| **STEP 2** <br/>(초기 데이터 확보) | 전담 매니저/기술팀 | **[데이터 수집 템플릿 배포 및 초기 입력]**: 표준화된 포맷과 함께 필수 질문 목록을 제공하고, 온라인 양식(Google Forms 등)으로 데이터를 받기 시작한다. | 1차 데이터 폼 작성 완료 건수 (KPI). |
| **STEP 3** <br/>(스토리 심층 인터뷰) | 전담 매니저/컨설턴트 | **[Storytelling Interview]**: 단순 정보가 아닌 '왜 이 일을 하는지', '어려움은 무엇인지' 등 감성적 요소를 파악하는 깊이 있는 대화 진행. (AOV 증진의 핵심). | 스토리 키워드 3개, 위기 극복 사례(스토리), 고유한 판매 포인트 문구. |
| **STEP 4** <br/>(데이터 검증 및 완성) | 영숙/플랫폼 기획팀 | **[크로스 체크 & 데이터 정제]**: 수집된 상품 정보와 스토리 데이터를 교차 검토하여, 상충되는 정보나 누락된 필드를 발견하고 생산자에게 피드백한다. | 최종 확정 데이터 세트(DB 업로드 가능 형태), 플랫폼 적용 가능한 태그 목록. |

### 📌 III. 표준화 데이터 포맷 (Standardized Data Template)

데이터를 **상품정보**, **생산 스토리**, **공급망 투명성** 세 가지 섹션으로 분리하여 수집해야 합니다. 이 구조가 플랫폼의 필터링과 콘텐츠 구현에 용이합니다.

#### 📊 A. [필수] 상품 기본 정보 (Product Core Data)
| 항목 | 데이터 유형 | 예시/설명 | 필수 여부 | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **상품명** | 텍스트 | 예: OO 마을 표고버섯 (참송이 등급) | ✅ | |
| **판매 단위** | 선택형 | 개 / 그램(g) / 세트 | ✅ | |
| **생산 지역** | 지역 코드/문자 | [강원도] OO군, XX면 | ✅ | LBS 연동 키워드 |
| **품질 등급** | 표준화 코드 | 참송이 1등급 (기준 명시) | ✅ | 가격 책정 근거 |
| **가격 정보** | 숫자/범위 | 최소 얼마 ~ 최대 얼마 (변동 가능성 포함) | ✅ | |

#### 🌱 B. [핵심] 생산 스토리 데이터 (Storytelling & Value Data)

| 항목 | 데이터 유형 | 예시/설명 | 필수 여부 | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **생산 주체의 철학** | 텍스트 (300자 내) | "저희 가족은 대를 이어 이 땅을 지키며..." | ✅ | 플랫폼 메인 스토리 섹션 활용 |
| **가장 큰 도전/위기 극복 경험** | 텍스트 (500자 내) | 기후변화, 유통시장 축소 등 어려움과 해결 과정. | ✅ | 신뢰 자본 증명 자료 |
| **고유한 판매 포인트(USP)** | 키워드/문장 | '오직 저희 마을에서만 나는 독특한 품종' | ✅ | 상품 설명의 핵심 강조 문구 |
| **소비자에게 전하는 메시지** | 텍스트 (100자 내) | "이 버섯은 건강을 넘어, 지역에 기여한다는 의미가 담겨있습니다." | ✨ | 구매 동기 부여 자료 |
