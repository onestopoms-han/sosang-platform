### 🎬 스토리보드 초안: '초기 유입 테스트 시나리오' (User Journey Map)

이 스토리보드는 **버전 1, 2, 3** 메시지가 어떤 흐름으로 사용자가 플랫폼을 경험하게 하는지를 보여줍니다.

| 단계 | 사용자 행동 (User Action) | 플랫폼 반응 (Platform Response) | 메시지/기능 | 기대 효과 |
| :--- | :--- | :--- | :--- | :--- |
| **1. 인지** | 검색창에 "소상공인 장부 프로그램" 입력 | 검색 결과 상단에 A/B 테스트 메시지가 노출됨 | **[헤드라인 + 본문 + CTA]** (버전별 랜덤 노출) | 높은 CTR 확보, 초기 관심 유도 |
| **2. 클릭** | CTA 버튼 클릭 → Landing Page 진입 | 페이지 로드 시 '3 분 만에 시작' 타이머 및 성공 사례 카운터 등장 | "성공한 사장님 1,000 명이 선택했습니다." (가상 데이터) | 신뢰도 형성, 체류 시간 증대 |
| **3. 탐색** | 주요 기능 (스마트 장부, AI 메뉴판) 클릭 시 데모 영상 자동 재생 | 데모 영상 속에서는 '재무 흐름 시각화' 및 '자동 디자인' 효과 강조 | **[가이드 툴]**: "이 버튼을 누르면 바로 시작됩니다." (간단한 버튼 위주의 UI) | 복잡한 기능 설명 없이 시각적 이해도 높임 |
| **4. 체험** | '무료 체험하기' 클릭 → 간단한 장부 입력 또는 이미지 업로드 | 실시간으로 결과물 생성 및 '재무 리포트 미리보기' 제공 | **PainGauge 로직**: 사용자의 입력 수준에 따라 맞춤형 가이드 제공 (예: 현금 흐름이 불안하면 재무 관리 팁) | 개인화 서비스 경험, 전환율 극대화 |
| **5. 전환** | 체험 후 "무료 플랜" 또는 "성공 사례 확인" 페이지 이동 | '지금 바로 시작하기' CTA 와 추가 혜택 (예: 1 년 무료 지원 컨설팅) 제시 | **[CTA 라이브러리]**: 여러 버전 중 전환율이 높은 것을 실시간으로 선택하여 노출 | 최종 목표 달성, 유료 플랜 또는 구독 유도 준비 |
