# 📊 외부 경제 지표 및 데이터 소스 리포트 (소상공인 리스크 관점)

**목표:** 대한민국 소상공인의 사업 리스크에 직접적으로 영향을 미치는 핵심 경제 지표를 선정하고, 신뢰할 수 있는 공공/민간 데이터 출처를 확보하여 플랫폼 연동을 위한 기초 데이터를 구축합니다.

## 1. 선정된 핵심 경제 지표 (Top 5)

소상공인의 재무적 손실과 심리적 불안정성(Loss Aversion & Anxiety)에 가장 직접적인 영향을 미치는 5가지 지표를 선정했습니다.

| 순위 | 핵심 지표 (Metric) | 소상공인 Pain Point 연관성 | 관련 감성 요소 |
| :--- | :--- | :--- | :--- |
| **1** | **기준금리 변동성 및 시장 금리** | 대출 이자 부담, 운영 자금 조달 비용의 직접적 영향. (재무적 손실) | 불안정성 (Anxiety), 재무적 압박 |
| **2** | **소비자 심리 지수 (Consumer Sentiment Index)** | 소비자의 구매 의향 변화를 예측하여 매출 변동성을 미리 감지. (예측 위험) | 불확실성, 희망/절망 |
| **3** | **물가 상승률 (소비자 물가 지수)** | 원자재 및 운영 비용의 급증으로 인한 마진 압박과 비용 통제 어려움. (손실 회피) | 불안정성, 손실 회피 |
| **4** | **고용 및 고용 시장 데이터 (취업률, 실업률)** | 인건비 부담 증가와 잠재적 수요 변화 예측의 근거. (운영 리스크) | 불안정성, 위협 |
| **5** | **특정 지역 상권 침체율 (지역별 소비 지표)** | 특정 지역 특화 사업자(식당, 소매점 등)에게 직접적인 매출 환경 변화를 반영. (지역적 리스크) | 고립감, 불안정성 |

## 2. 데이터 출처 및 API 목록

선정된 각 지표에 대해 신뢰도와 접근성을 기준으로 공공기관 및 주요 금융 데이터를 중심으로 조사했습니다.

| 지표 | 추천 데이터 소스 (API/DB) | 제공 데이터 형태 | 신뢰도 평가 | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **1. 기준금리 변동성** | 한국은행 경제통계시스템 (ECOS) API / 한국은행 금융통화위원회 자료 | 금리, 기준금리 변화 추이 | 최상 (공식 통계) | 실시간/주기별 데이터 확보 용이. |
| **2. 소비자 심리 지수** | 한국은행 경제연구원 / KDI (한국개발연구원) 데이터베이스 | 소비자 신뢰 지수, 소비 지출 동향 | 상 (정부 기관) | 월별/분기별 데이터 제공. |
| **3. 물가 상승률** | 통계청 국가통계포털 (KOSIS) / 한국은행 | 소비자물가지수(CPI), 생산자물가지수(PPI) | 최상 (공식 통계) | 핵심 인플레이션 지표. |
| **4. 고용 시장 데이터** | 통계청 경제활동인구조사 | 실업률, 고용률, 산업별 고용 동향 | 최상 (정부 통계) | 소상공인의 인건비 및 수요 예측에 필수. |
| **5. 지역 상권 침체율** | 통계청 지역별 사업체 등록 및 변화 데이터 / 지방자치단체 통계 | 특정 시/군/구 단위의 업종별 신규/폐업률, 소비 지출 추이 (DB 연동 필요) | 중상 (데이터 통합 필요) | **[추측]** 실시간 API는 어려우므로, 공공 DB를 주기적으로 크롤링하거나 별도 데이터 제공처 탐색 필요. |

## 3. 연구 결과 요약 및 플랫폼 적용 방향성

### A. 데이터 신뢰도 확보 방안
*   **최우선 채널:** 한국은행(BOK), 통계청(KOSTAT), KDI 등 정부 기관의 공식 데이터를 최우선 소스로 설정하여 데이터의 객관성과 신뢰도를 극대화합니다.
*   **플랫폼 적용:** 모든 지표는 'PLRI(잠재적 리스크 감소)' 모델의 입력 변수로 사용되며, 단순히 수치를 보여주는 것이 아니라, **"이 지표가 X% 변동할 경우 예상되는 재무적 손실 범위와 필요한 행동 계획(Actionable Blueprint)"**을 함께 제시하도록 설계합니다.

### B. PLRI/AI 시스템 연동 제안
*   **예측 모델 강화:** 단순 과거 데이터 분석에서 벗어나, 실시간으로 수집된 외부 경제 지표를 기반으로 **'위기 감지 경보 시스템(Warning System)'**을 구축하여 소상공인에게 선제적인 통제권을 부여해야 합니다. (Researcher 검증된 지식: 예측 중심 $\rightarrow$ 실행 중심)
*   **감성적 UX:** 재무적 변동성(①)과 심리적 불안정성(③)의 상관관계를 시각화하여, 소상공인이 데이터를 통해 **'불안을 감소시키고 통제권을 회복하는 경험'**을 제공하도록 UI/UX를 설계해야 합니다.
