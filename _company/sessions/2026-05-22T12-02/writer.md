## ✍️ Dynamic Messaging Template (DMT) 기획 및 작성

랜딩 페이지의 핵심 전환점(Touchpoints)에 맞춰, 데이터-디자인 정합성 프레임워크를 기준으로 **상황별 카피 세트**를 작성했습니다. 이 내용은 디자이너가 시각적 요소로 구현하고, 개발자가 조건문(`if/else`)으로 제어할 수 있도록 명확히 분리되었습니다.

### 📍 Context: 핵심 CTA 섹션 (ROI/Action Plan 제시 영역)
*   **전제:** 사용자가 플랜을 선택하거나 진단을 받으면, `estimated_saving` 값이 산출되고 이 값을 기반으로 카피가 변화해야 합니다.

#### 1. [Success State] — 데이터 절감액이 크게 나타났을 때 (Loss 최소화 성공)
*   **조건:** `estimated_saving` > $500 AND `success_status` == 'SUCCESS'
*   **목표 카피 역할:** 감동(Emotional Impact)과 즉각적인 행동 유도.

| 요소 | 카피 내용 | 마크다운/스타일링 지침 (Designer/Developer에게 전달할 지침) |
| :--- | :--- | :--- |
| **메인 메시지** | "✅ 지금 당장 아낄 수 있는 비용이 이렇게 컸습니다." | 배경을 'Growth Green' 계열로 강조. 헤드라인 크기 최대로 설정. |
| **데이터 시각화 문구** | *당신의 예상 손실 비용: $X,XXX* → **BDS 솔루션 적용 후 절감액: $Y,YYY** *(절감률: Z%)* | 원본과 비교하는 대비 효과를 극대화 (Strike-through 처리). 숫자에 볼드 처리를 강력하게. |
| **보조 카피 (설득)** | "이 금액은 단순한 비용 절감이 아닙니다. 폐업 위기에서 다시 일어날 수 있는 '새로운 기회'에 대한 투자입니다." | 신뢰감 있고, 인간적인 어조로 전환점을 강조하는 문구 배치. |
| **CTA 버튼 텍스트** | "나의 맞춤형 성장 플랜 받아보기 (무료 진단)" | Primary CTA 색상 유지. 가장 눈에 띄게 처리. |

#### 2. [Neutral State] — 데이터 절감액이 적거나 애매할 때 (정보 부족)
*   **조건:** `estimated_saving` < $100 OR `success_status` == 'PENDING'
*   **목표 카피 역할:** 실망감을 느끼지 않게 하면서, 다음 정보를 얻도록 유도.

| 요소 | 카피 내용 | 마크다운/스타일링 지침 |
| :--- | :--- | :--- |
| **메인 메시지** | "🤔 더 정확한 진단을 위해 몇 가지 추가 정보가 필요합니다." | 색상을 중립적인 톤(Warm Gray)으로 설정하여 압박감을 낮춤. |
| **보조 카피 (설득)** | "저희 시스템은 소상공인의 사업 구조를 심층 분석해야 가장 큰 절감 효과를 찾을 수 있습니다. 잠시만 시간을 내어 [A]와 [B] 정보를 제공해주시겠어요?" | 질문의 형태를 취하여 사용자의 참여(Engagement)를 요청. |
| **CTA 버튼 텍스트** | "추가 진단 정보 입력하기" (→ 다음 단계로 유도하는 서브폼으로 연결) | Primary CTA보다 낮은 우선순위의 Secondary 버튼 디자인 적용. |

#### 3. [Failure State] — 시스템이 해결책을 제시하지 못했을 때 (예외 처리)
*   **조건:** `success_status` == 'FAILED' OR API 호출 오류 발생
*   **목표 카피 역할:** 실패로 인한 좌절감을 방지하고, BDS 플랫폼의 인간적인 지원(Consulting)으로 전환.

| 요소 | 카피 내용 | 마크다운/스타일링 지침 |
| :--- | :--- | :--- |
| **메인 메시지** | "🥲 당장 해결책을 찾기 어려우시더라도, 저희가 함께 고민하겠습니다." | 진심이 담긴 사과와 공감의 톤앤매너 사용. (따뜻한 색상 계열) |
| **보조 카피 (설득)** | "지금은 데이터만으로는 부족합니다. 소상공인 대표님의 '경험'과 '이야기'가 필요할 때입니다. 저희 전문가 팀이 직접 이야기를 듣고, 플랜을 재구성해 드릴게요." | 제품 판매에서 '컨설팅 서비스(BDS)'의 가치로 전환하는 문구 사용. (플랫폼이 아닌 사람에게 의지하게 만듦) |
| **CTA 버튼 텍스트** | "전문가 상담 신청하기" (→ 문의/상담 예약 폼으로 연결) | 가장 낮은 단계의 CTA이지만, 회사의 본질적인 가치를 전달하는 핵심 액션임을 강조. |
