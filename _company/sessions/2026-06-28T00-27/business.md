## 3. 기술 실행 계획과의 불일치 확인

- **시각화 가이드:** Designer 가 "90% AI 정확도"를 시각화하되, 신뢰 구간 (예: 85~92%) 을 함께 표시해야 합니다. → **Designer 로 전달: 신뢰 구간을 포함한 차트 배치안을 수정하라.**

## 4. 시장 데이터 기반 ROI 계산식 예시 (검증용)
```python
# Python: 실제 ROI 계산 시 사용 가능한 예제 코드
import pandas as pd

def calculate_roi(monthly_growth_rate, initial_revenue, months):
    final_revenue = initial_revenue * ((1 + monthly_growth_rate) ** months)
    absolute_increase = final_revenue - initial_revenue
    roi_percent = (absolute_increase / initial_revenue) * 100
    return absolute_increase, roi_percent

# 가정: 초기 월 매출 1 천만 원, 3 개월 성장률 5%, 최종 수익 증가율 계산
initial_revenue = 1e7   # 1 억 원
monthly_growth_rate = 0.05  # 5%
months = 3
abs_increase, roi_percent = calculate_roi(monthly_growth_rate, initial_revenue, months)

print(f"초기 매출: {initial_revenue:,} 원")
print(f"3 개월 후 매출: {(initial_revenue * ((1 + monthly_growth_rate) ** months):,} 원")
print(f"증가액: {abs_increase:,} 원")
print(f"ROI (증가율): {roi_percent:.2f}%")
```
