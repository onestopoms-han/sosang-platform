| 구분 | 유튜브 숏폼 평균 | 인스타그램 릴스 평균 | 본 프로젝트 목표 |
|------|------------------|---------------------|-----------------|
| **초기 지속률** | 45~60 초(8~12%) | 30~40 초(5~8%) | 90 초 이상(15% 이상) |
| **상위 10%** | 70 초 이상(15~20%) | 50 초 이상(12~18%) | 120 초 이상(25% 이상) |

## 2. 검증된 후크 패턴 (신뢰·위기 감정 매핑 적용)
| 후크 ID | 내용 예시 | 감정 변화 | 예상 지속률 | 시각화 요구사항 |
|---------|-----------|------------|--------------|-----------------|
| **H01** | "매출이 안 오를 때… ‘그냥 더 열심히’ 하셨죠?" (위험 신호) → 불안 → 해결 제시 | 🟥→🟨→🟢 | 15~20% | Trust Widget 경고 아이콘 + PainGauge 노란색 |
| **H02** | "당신의 가게, 안전한가요?" (질문형) → 호기심 자극 → 데이터 제시 | 🟨→🔵 | 18~22% | Trust Widget 파란색 포인트 + KPI 대시보드 |
| **H03** | "이 공식은 매출 불안 끝내주는 법" (솔루션 중심) → 신뢰 형성 → 행동 유도 | 🟢→🔵 | 20~25% | PainGauge 초록색 전환 + CTA 그래픽 |

# 🎯 플랫폼별 CTA 최적화 전략 가이드 (V1.0)

## 1. 행동 유도 메시지 (CTA) 전환율 극대화 원칙
| 구분 | 핵심 목표 | 최적화 포인트 |
|------|-----------|----------------|
| **유튜브** | 베타 참여 링크 클릭 유도 | "베타 테스트에 참여하세요" → "이 데이터를 기반으로 실행 가능한 다음 스텝을 찾아보세요" (구체적 행동 제시) |
| **인스타그램** | 저장/공유 및 CTA 버튼 활용 | "안전 마진 80% 달성하는 사장님" → "이 카드는 당신의 생존 전략입니다. 공유하세요!" (가치 제공 강조) |
| **블로그** | 이메일 구독 및 콘텐츠 공유 | "소상공인의 AI 리스크 대시보드 가이드를 읽어보세요" → "구독하면 다음 팁을 먼저 확인하세요" (유지된 관심 유도) |

## 2. CTA 메시지 변형 전략 (A/B 테스트 기반)
| 변형 ID | 예시 메시지 | 예상 전환율 | 적용 시기 |
|---------|-------------|--------------|-----------|
| **CTA01** | "베타 참여 링크: [주소]" | 8~12% | 영상 종료 시점, 피드 첫 줄 |
| **CTA02** | "이 데이터가 당신의 다음 스텝을 알려줍니다" | 15~20% | 솔루션 제시 후, 공감 유도 |
| **CTA03** | "안전 마진 공식: [링크]" | 20~25% | 해결책 제시 시점, 구체적 행동 유도 |

## 3. 시각·문자 결합 전략 (Trust Widget/PainGauge 적용)
- **유튜브:** CTA 화면에 Trust Widget 파란색 포인트 + "지금 참여하세요" 텍스트 오버레이  
- **인스타그램:** 카피스트릴에 PainGauge 초록색 전환 시점 동기화 → "이제 당신의 차례입니다" 메시지 추가  

## 4. 즉시 실행 체크리스트
- [ ] 후크 ID 선택 및 감정 매핑 계획 수립  
- [ ] CTA 변형 3 개 생성 (각 플랫폼별 최적 조합)  
- [ ] 시각적 일관성 (Trust Widget/PainGauge) 검증  
- [ ] 배포 전 A/B 테스트 결과 예측
