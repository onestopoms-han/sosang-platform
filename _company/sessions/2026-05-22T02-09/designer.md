## 📝 Action Plan 최종 인터페이스 디자인 사양서 (Final UI/UX Specification)

'우선순위 명확화'와 'Pain Point 연결 강화'를 핵심 가치로 설정하여, 사용자가 정보를 소비하는 과정에서 즉각적인 행동 변화(Action)를 유도하도록 설계합니다.

### 1. 핵심 디자인 원칙 (Guiding Principles)

| 원칙 | 설명 | 시각적 구현 목표 |
| :--- | :--- | :--- |
| **Pain Point First** | 모든 액션은 사용자가 현재 겪고 있는 가장 큰 고통(Pain Point)에서 시작해야 함을 명시. | Pain Point는 항상 최상단에 배치하며, 그 아래에 해결책(Action Item)이 위치하도록 계층 구조 설정. |
| **Prioritization Clarity** | 복잡한 정보 속에서 사용자가 무엇을 먼저 해야 할지 즉각적으로 인지하게 함. | '우선순위'를 시각적 계층(Hierarchy)과 색상으로 명확히 구분하여, 가장 중요한 항목에 집중하도록 유도. |
| **Effort Reduction** | 액션 수행에 필요한 단계를 최소화하고, 불필요한 탐색을 제거함. | 'Basic Action'은 드롭다운이나 플로우 차트 형태로 시각적으로 간결하게 표현. |

### 2. 최종 레이아웃 및 컴포넌트 구조 (Layout & Component Structure)

**[Action Plan View]**는 크게 **Pain Point 섹션**과 **Solution/Action 섹션**으로 나뉩니다.

#### A. 상단: Pain Point 연결 강화 영역 (The Hook)
*   **목표:** 사용자가 현재의 어려움을 즉시 인식하고 공감하게 만듦.
*   **구성:** 카드 또는 탭 형태로, 가장 강력한 **Pain Point Trigger**를 강조합니다.
    *   **디자인:** 배경색을 미묘한 경고 색상(예: 연한 주황/노랑)으로 설정하여 긴급성을 부여합니다.
    *   **요소:** 각 Pain Point는 클릭 시 해당 액션 플랜 전체로 이동하거나, 상세 설명 팝업을 띄우도록 설계합니다.

#### B. 중앙: 우선순위 명확화 영역 (The Core Flow)
*   **목표:** AI가 제안한 Action Item들을 사용자가 쉽게 순위를 매기고 실행할 수 있도록 함.
*   **구성:** **우선순위 박스(Prioritization Box)**를 핵심 요소로 사용합니다.
    1.  **상위 레벨 (P1/P2):** 가장 중요한 액션 플랜을 명확히 구분하는 굵은 시각적 분리선 또는 색상 코드를 적용합니다. (예: P1은 진한 파란색, P2는 연한 파란색)
    2.  **Action Item 카드:** 각 Action Item은 다음 정보를 포함하며, 사용자가 드래그 앤 드롭으로 순위를 변경할 수 있도록 합니다.
        *   `Area` (문제 영역): 아이콘과 함께 명확하게 표시.
        *   `Pain Point Trigger`: 연결된 Pain Point를 하이라이트하여 시각적 연결을 강화합니다.
        *   `Basic Action`: 간결한 텍스트와 함께, 실행 버튼(CTA)을 최소화하고 플로우로 유도합니다.

#### C. 하단: 결과 및 피드백 영역 (The Feedback Loop)
*   **목표:** 액션 수행 후의 성과를 추적하게 하여 동기를 부여함.
*   **구성:** **Progress Tracker** 시각화.
    *   **디자인:** 전체 Action Plan 대비 완료된 항목의 비율을 원형 또는 진행 막대로 표시합니다. (시행착오 감소에 기여했음을 직관적으로 보여줌)

### 3. 컬러 및 타이포 제안 (Color & Typography Suggestion)

| 요소 | 제안 값 | 이유 |
| :--- | :--- | :--- |
| **Primary Color** | `#4A90E2` (BDS 브랜드 블루) | 신뢰감과 전문성을 제공하며, AI 기반의 논리적 시스템을 시각적으로 표현. |
| **Pain Point Highlight** | `#F5A623` (Amber/Orange) | 경고 및 주의를 나타내어 사용자의 즉각적인 주목을 유도하고 긴급성 부여. |
| **Priority Level (P1)** | `#337AB7` (Darker Blue) | 가장 높은 우선순위를 명확히 구분하여 시각적 계층 구조를 확립. |
| **Typography** | Noto Sans KR 또는 Pretendard | 가독성이 높고, 데이터 중심의 정보 전달에 적합한 산세리프 폰트 사용. |
