# 🎬 숏폼 영상 스토리보드: Trust Widget/PainGauge 시각화 효과 극대화 (V1.0)

**작성일:** 2026.06.28  
**목표:** 시청자 3 초 내 불안감 → 공감 → 해결책 유도 (신뢰도/위기감 가시화)  
**참고 디자인 시스템:** Designer V3.0 색상 매핑 (Trust: 🟢, Warning: 🟡, Critical: 🔴)

---

## 🎯 핵심 전략
- **색상 = 감정:** 화면 전체에 Trust Widget/PainGauge 색상이 스며들도록 필터와 오버레이 적용.
- **시각적 은유:** '안전 마진' → 🟢 초록색 영역 확대, '불안한 예측' → 🔴 빨간색 필터로 흔들리는 화면.
- **타이밍:** 0~3 초 강렬 후크 (빨강) → 3~15 초 공감 (노랑/초록 전환) → 15~30 초 해결책 (파랑) → CTA.

## 🛠️ 제작 프로세스 가이드라인

### 1. 촬영 및 편집 준비
- **장소:** 사장님 실제 가게, 혹은 골목상권 배경이 보이는 장소.
- **조명:** 
    - 후크 장면: 빨간색 LED 스트립 또는 필터 (Critical) 적용.
    - 공감 장면: 노란색 조명 (Warning) 은유적 효과.
    - 해결책 장면: 자연광 또는 파란색 조명 (Solution) 사용.
- **편집 도구:** 
    - **CapCut / Premiere Pro** 템플릿 미리 준비.
    - **필터 설정:** `LUT` 파일로 색상 일관성 확보 (Trust/Warning/Critical LUT).

### 2. 시각적 효과 적용
- **오버레이:** 
    - 화면에 Trust Widget/PainGauge 이콘을 오버레이로 배치 (화면 흔들림과 대비).
    - 색상에 따른 필터 적용 (Critical: 🔴, Warning: 🟡, Trust: 🟢, Solution: 🔵).
- **텍스트 스타일:** 
    - Critical: 빨간색 굵은 폰트.
    - Warning: 노란색 중간 굵기.
    - Trust/Solution: 초록/파랑색 세련된 폰트.

### 3. 오디오 믹싱
- **BGM:** 
    - 후크: 심장이 뛰는 소리 (심장 박동음).
    - 공감: 불안한 리듬 → 해결책: 밝은 비트로 전환.
- **음성:** 
    - 내레이터: 차분하고 신뢰감 있는 톤 (남성/여성 모두 가능).
    - 자막: 화면 중앙 또는 하단에 배치, 색상별 폰트 적용.

# 📋 숏폼 제작 프로세스 가이드라인 (V1.0)

**목적:** Trust Widget/PainGauge 시각화 효과를 극대화한 숏폼 영상 제작을 위한 실무 지침  
**참고 문서:** `youtube_storyboard_v1.md`  

---

## 🎯 작업 흐름 (Work Flow)

### 1. 촬영 준비 (Pre-Production)
- **장소 선정:** 
    - 골목상권 배경이 보이는 장소 또는 사장님 실제 가게 내부.
    - 자연광 활용 (신뢰감 전달).
- **조명 설정:** 
    - Critical: 빨간색 LED 스트립 또는 필터 사용 (화면 흔들림 효과).
    - Warning: 노란색 조명 은유적 효과.
    - Trust/Solution: 자연광 또는 파란색 조명 (안정적 분위기).
- **주행 촬영:** 
    - 사장님 인터뷰, 가게 운영 모습, 데이터 대시보드 시연 (실제 화면 캡처 활용).

### 2. 편집 및 시각화 (Production)
- **편집 도구:** CapCut / Premiere Pro 템플릿 사용.
- **필터 및 LUT 적용:** 
    - Critical: 🔴 빨간색 필터 (`brightness: 0.6`, `contrast: 1.2`).
    - Warning: 🟡 노란색 필터 (`brightness: 0.8`, `contrast: 1.1`).
    - Trust: 🟢 초록색 필터 (`brightness: 1.0`, `contrast: 1.0`).
    - Solution: 🔵 파란색 필터 (`brightness: 1.1`, `contrast: 0.9`).
- **오버레이 및 이콘:** 
    - 화면 중앙 또는 하단에 Trust Widget/PainGauge 이콘 배치 (화면 흔들림과 대비).
    - 색상별 폰트 적용 (Critical: 빨간색 굵은 폰트, Warning: 노란색 중간 굵기 등).

### 3. 오디오 믹싱 (Post-Production)
- **BGM:** 
    - 후크: 심장이 뛰는 소리 (심장 박동음).
    - 공감: 불안한 리듬 → 해결책: 밝은 비트로 전환.
- **음성:** 
    - 내레이터: 차분하고 신뢰감 있는 톤.
    - 자막: 화면 중앙 또는 하단에 배치, 색상별 폰트 적용.
