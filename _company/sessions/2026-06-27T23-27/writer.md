- **초기 투자금 가정**: ₩5 억 (플랫폼 개발, 마케팅, 운영 인력 등)

### 가. 초기 투자 구조 (Year 0)
| 항목 | 비용 (₩) | 비고 |
| :--- | :--- | :--- |

| 마케팅 예산 (초기) | 2,000 억 | 베타 참여자 모집, 인플루언서 협업 |
| 운영 인력 (1 년분) | 800 억 | 개발팀, CS 팀, 마케팅 팀 등 |
| 예비비 | 700 억 | 리스크 대비 |
| **총 투자** | **5,000 억** | |

### 나. 연간 매출 성장 예측 (Revenue Projection)
| 해 | 예상 매출액 (₩ 억) | 성장률 (%) | 비고 |
| :--- | :--- | :--- | :--- |
| 1 차년도 | 8,000 억 | - | 베타 참여자 기반 확장 |
| 2 차년도 | 15,000 억 | 87.5% | 데이터 기반 마케팅 효과 극대화 |

### 다. 비용 구조 및 순이익 (Cost Structure & Net Profit)
| 항목 | Year 1 | Year 2 | Year 3 | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| 매출액 | 8,000 억 | 15,000 억 | 30,000 억 | |
| COGS (인건비 등) | 2,400 억 | 4,500 억 | 9,000 억 | 매출액의 30% |
| 운영비 (마케팅 등) | 1,500 억 | 2,500 억 | 4,500 억 | 매출액의 18.75% |
| 총 비용 | 3,900 억 | 7,000 억 | 13,500 억 | |
| **순이익** | **4,100 억** | **8,000 억** | **16,500 억** | |

### 라. ROI 계산 (Investment Return Calculation)
- **초기 투자금**: 5,000 억
- **3 년 차 누적 순이익**: 4,100 억 + 8,000 억 + 16,500 억 = 28,600 억
- **누적 ROI**: (28,600 억 - 5,000 억) / 5,000 억 × 100% = 472%
- **3 년 차말 ROI (단기 기준)**: 16,500 억 (Year 3 순이익) / 5,000 억 = 330%

> **주**: 3 차년도 말에만 투자금의 3 배 이상 수익을 내므로, '3 년 내 ROI 200%'는 Year 3 말에 달성됨.
> 또한 Year 1~2 의 순이익이 5,000 억 을 상회하므로, 실제 현금 흐름은 더 빠름.
