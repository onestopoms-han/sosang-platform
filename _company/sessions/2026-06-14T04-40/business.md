## 🎯 목표: 데이터 추적 로직 (IP-01~IP-04) 과 수익 전환 지점의 최종 연결

## 1️⃣ 데이터 추적 로직과 KPI 의 연결: IP-01~IP-04 의 수익화 포인트
### **IP-01 (사용자 행동 기반 심층 관심도)**
- **수익 전환 지점**: 랜딩 페이지에서 '심층 관심도'가 높은 사용자는 '진단 시작' 버튼 클릭률이 3 배 높습니다.
- **실행 전략**: IP-01 로 계산된 점수가 높은 사용자에게는 '무료 진단' 대신 '유료 프리미엄 분석'을 추천합니다 (Upsell Opportunity).
- **목표치**: 심층 관심도 점수 > 70 일 때 유료 전환율 목표 = 5%.

### **IP-02 (PainGauge 점수와 위험 인지도)**
- **수익 전환 지점**: PainGauge 점수가 높을수록 '진단 결과'의 신뢰도가 높아지며, 이는 '솔루션 구매'로 이어집니다.
- **실행 전략**: IP-02 로 계산된 '위험 인지도'가 80 이상일 때만 'AI 컨설팅 유료화'를 제안합니다 (High-intent Targeting).
- **목표치**: 위험 인지도 > 80 일 때 솔루션 구매 전환율 목표 = 15%.

### **IP-03 (해결 의지 측정)**
- **수익 전환 지점**: 진단 페이지에서 '솔루션 클릭률'이 높을수록 '구독/구매'로 이어집니다.
- **실행 전략**: IP-03 로 계산된 '해결 의지'가 60 이상일 때 '무료 샘플' 대신 '유료 솔루션'을 제안합니다 (Value Demonstration).
- **목표치**: 해결 의지 > 60 일 때 유료 전환율 목표 = 10%.

### **IP-04 (신뢰도 및 구매 전환)**
- **수익 전환 지점**: 최종 로드맵 다운로드/구독 시 '신뢰도'가 높을수록 '유료 컨설팅 예약'으로 이어집니다.
- **실행 전략**: IP-04 로 계산된 '신뢰도' 점수가 90 이상일 때만 '1:1 유료 컨설팅'을 제안합니다 (High-Ticket Offer).
- **목표치**: 신뢰도 > 90 일 때 유료 컨설팅 전환율 목표 = 20%.

## 2️⃣ 개발팀 액션: 백엔드 API 스펙 및 데이터 구조 정의
### **필요한 API 엔드포인트**
1.  `/api/v1/user/behavior-log`: 사용자 행동 로그 (클릭, 스크롤) 수집.
    - **Payload**: `{ user_id, event_type, timestamp, page_url, scroll_depth }`
2.  `/api/v1/pain-gauge/calculate`: PainGauge 점수 계산 및 위험 인지도 측정.
    - **Payload**: `{ user_data, diagnostic_results }` (Post-processing 로직 필요)
3.  `/api/v1/trust-widget/update`: Trust Widget 점수 업데이트 및 신뢰도 시각화 데이터 제공.
    - **Payload**: `{ trust_score, last_update_timestamp }`

### **데이터 수집 규칙**
- **PainGauge 점수 계산**: 사용자 입력 데이터를 기반으로 '위험 인지도'를 실시간으로 계산하여 IP-02 로 사용.
- **Trust Widget 업데이트**: 진단 결과를 기반으로 '신뢰도 점수'를 계산하여 IP-04 로 전달.

---

## 3️⃣ 디자인/마케팅팀 액션: 콘텐츠 전략 및 전환 포인트 정의
### **'위험 → 해결 → 성과' 여정별 메시지 규칙**
- **STEP 1 (위험 인지)**: PainGauge 점수가 높을수록 '무료 진단' 대신 '유료 프리미엄 분석'을 제안합니다. (Upsell Opportunity)
- **STEP 2 (해결 제시)**: Trust Widget 점수가 높을수록 'AI 컨설팅 유료화'를 제안합니다. (High-intent Targeting)
- **STEP 3 (성과 기대)**: IP-04 로 계산된 '신뢰도'가 높을 때 '1:1 유료 컨설팅'을 제안합니다. (High-Ticket Offer)

### **최종 로드맵 페이지 전략**
- **신뢰도 점수 > 90**: '1:1 유료 컨설팅 예약' 버튼 노출.
- **신뢰도 점수 70~89**: '유료 솔루션 구독' 제안.
- **신뢰도 점수 < 70**: '무료 샘플 다운로드' 유도.
