# ✍️ Writer — Designer가 확정한 UI/UX 흐름에 맞춰, AI 진단 결과에 기반한 실제 사용 사례와 'Action Prescription'의 명확성을 극대화하는 심층 지식 콘텐츠 초안을 보강하라.

✍️ Writer: 작업 시작합니다.

CEO의 지시와 현재 Designer가 확정한 UI/UX 흐름(진단 → 행동 제시)을 종합적으로 고려했을 때, 가장 시급하고 가치 있는 단일 작업은 **AI 진단 결과를 실제 소상공인이 이해하고 실행할 수 있도록 구조화한 '지식 아키텍처'를 설계**하는 것입니다.

단순히 지식을 나열하는 것이 아니라, 진단의 실패 지점(Pain Point)을 콘텐츠의 후크로 활용하여, 즉각적인 행동 변화(Actionable Insight)를 유도하는 심층 학습 모듈을 만들어야 합니다.

이 작업을 위해 **Writer가 아키텍처 설계**를 주도하고, **Designer에게는 해당 구조에 맞는 인터랙티브 컴포넌트 추가 검토**를 요청합니다.

---

### 📝 작업 산출물: AI 진단 기반 '액션 처방(Action Prescription)' 심층 지식 콘텐츠 아키텍처 (V1.0)

이 문서는 Designer가 확정한 리포팅 페이지의 핵심 섹션인 **'Action Prescription'** 영역에 들어갈 모든 교육용 콘텐츠의 구조와 흐름을 정의합니다.

#### 🎯 1. 콘텐츠 목표 및 가치 정의
*   **최종 사용자 경험:** 진단 결과(예: "온라인 노출 점수 하락")를 보고 좌절하는 대신, "이것은 문제가 아니라 개선할 수 있는 영역이다"라는 동기 부여와 함께 **구체적인 다음 스텝을 얻어간다.**
*   **핵심 기능:** 콘텐츠는 이론 설명에 그치지 않고, 💡 **'개념 이해 → 실패 원인 분석(Diagnosis) → 실습 가이드(Action Plan) → 성공 사례(Use Case)'**의 흐름을 따라야 합니다.

#### 📐 2. '액션 처방' 지식 콘텐츠 모듈 구조 (Module Template)
진단 결과 항목 하나를 깊이 있게 설명할 때마다 아래와 같은 표준화된 **지식 모듈(Knowledge Module)**을 반복 적용합니다. 이 구조가 전체 리포팅 페이지의 전문성을 결정합니다.

| 단계 | 컴포넌트 명칭 | 콘텐츠 내용 (Writer 역할) | Designer 요청 사항 | 목표 효과 |
| :--- | :--- | :--- | :--- | :--- |
| **A. 문제 인식** | **[진단 항목 상세 분석]** | *AI가 발견한 실패 원인*을 소상공인의 언어로 풀어서 설명. (예: "우리 가게의 지역 키워드 활용이 부족합니다.") | 진단 점수와 함께 표시되는 **'위험도/개선 필요 레벨'** 배지 구현. | 문제 공감 및 경각심 유발. |
| **B. 개념 학습** | **[핵심 원리 이해: 이론]** | 해당 분야의 기초 지식 설명 (전문 용어 해설 포함). *왜* 이 활동이 중요한지 논리적으로 설득. | 짧은 애니메이션 또는 인포그래픽 삽입 공간 확보. (시각적 이해도 증대) | 전문성 부여 및 학습 욕구 자극. |
| **C. 실행 계획** | **[3단계 액션 플랜: 실습]** | 가장 중요한 부분. 원론을 실제 행동으로 전환하는 3~5가지의 구체적인 체크리스트 제시. (**가장 높은 ROI를 가진 활동 1순위 강조**) | **'체크리스트(Checklist)'** 컴포넌트 구현 (사용자가 직접 클릭하여 진도 표시 가능). | 즉각적인 실행 동기 부여 및 행동 유발. |
| **D. 성공 사례** | **[성공 스토리: Use Case]** | 유사 업종/지역의 성공 사례를 제시. '무엇을 했더니(Action) → 어떤 결과가 나왔는지(Result)' 구조로 서술. (익명화 필수) | Before & After 이미지 또는 간결한 통계 그래프 삽입 공간. | 신뢰성 확보 및 동기 부여 극대화. |

#### 🌐 3. 적용 예시: '지역 온라인 노출 점수'가 낮을 경우
| 모듈 섹션 | 콘텐츠 초안 (Writer) |
| :--- | :--- |
| **A. 문제 인식** | "🚨 지역 키워드 활용 취약! 고객이 우리 가게를 찾기 어려운 이유입니다."<br>당신의 온라인 노출은 '거대한 지도'에서 길을 잃은 것과 같습니다. (진단 근거: 경쟁사 대비 리뷰 수/키워드 빈도율 등) |
| **B. 개념 학습** | **[지역 검색 엔진의 작동 원리]** 지역 키워드는 단순히 가게 이름을 넣는 것이 아닙니다. '동네 맛집', 'OO역 근처 밥집'처럼 고객이 실제로 사용하는 구어체 조합을 이해하는 것이 핵심입니다. (설명: 네이버/카카오 로컬 SEO 개념). |
| **C. 실행 계획** | **✅ Action Plan: 3단계로 온라인 노출력을 높이는 법**<br>**STEP 1 (기초):** 지도 등록 정보(영업시간, 주차장)를 *최신화*하고 누락된 사진 5장을 업로드하세요.<br>**STEP 2 (핵심):** 네이버 플레이스에 '지역 키워드'가 포함된 메뉴 설명과 후기를 월 1회 이상 작성하는 습관을 만드세요. (예: "OO역 근처 데이트 코스로 좋아요.")<br>**STEP 3 (확장):** 인스타그램 하이라이트에 '우리 동네 인기 조합' 같은 지역 특화 해시태그를 필수로 포함하세요. |
| **D. 성공 스토리** | **🏆 [사례] OO 카페 사장님의 온라인 노출력 개선기**<br>**(Before)**: 단순히 가게 사진만 올림. 검색 시 노출률 5%.<br>**→ (Action):** 매주 지역 맛집 커뮤니티에 '오늘의 추천 메뉴'로 글을 쓰고, 키워드를 활용하여 후기를 남김.<br>**✅ (After):** 3개월 만에 온라인 노출률 40% 상승. 월 방문객 증가율 **25%**. |
