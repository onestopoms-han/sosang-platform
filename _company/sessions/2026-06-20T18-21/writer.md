# BDS 소상공인 플랫폼 - 감성 스토리텔링 및 교육 자료 기획서 (Draft V1.0)

## 1. 서론: 기술과 감성의 교차점
**배경:** 기존 소상공인 솔루션은 '데이터'와 '기능'에 치중했으나, 실제 사용자 경험에서는 불확실성과 불안감이 존재했습니다. BDS 플랫폼의 핵심 가치인 **Trust Widget (신뢰도 시각화)** 과 **PainGauge (핵심 애로 지표 파악)** 은 이러한 감정적 장벽을 데이터 기반 솔루션으로 해결합니다.

**목표:**
- 소상공인의 불안한 심리 상태를 '시각화'하여 이해시키고, 이를 플랫폼의 기능을 통해 해소하는 과정을 스토리로 풀어냄.
- 마케팅·교육 자료를 위한 구체적인 콘텐츠 구조 (후크, 본문, CTA) 를 정의함.

---

## 2. 감성 스토리텔링 초안: "데이터가 만드는 안식처"

### [Act 1: 불안의 시작] - "내 가게, 누구도 알지 못해"
- **상황:** 새벽 4 시, 가게 문을 닫고 잠자리에 들지만 걱정만 남음. "오늘 매출이 떨어졌는데 그 이유는?" "고객들이 왜 우리 제품을 안 사요?"
- **감정적 연결점 (Pain Point):** 불확실성, 두려움, 고립감.
- **시각적 메타포:** 어두운 방 안에 떠다니는 작은 불빛들 (데이터 포인트).

### [Act 2: 만남] - "나를 도와줄 파트너가 있다고?"
- **발생:** SNS 에서 'BDS 플랫폼'이라는 이름의 아이콘을 발견. 하지만 기능 설명만으로는 부족함.
- **기술적 접점 (Trust Widget):** "안녕하세요, 당신의 가게 데이터를 분석해 드립니다."라는 첫인사 대신, **"오늘의 신뢰도 85%"** 같은 구체적인 수치를 보여주는 인터랙티브 UI 를 경험함.
- **감정 변화:** '어려운 기술'이 아니라 '나를 이해해주는 친구'로 인식 전환 시작.

### [Act 3: 실행] - "내 가게, 이제 내가 다스려요"
- **상황:** 플랫폼의 PainGauge 도구를 통해 "고객들이 가장 관심 있는 포인트는 가격 대비 품질이에요"라는 인사이트를 얻음.
- **기술적 접점 (Action Prescription):** 단순히 데이터를 보여주기만 하지 않고, **"이제 메뉴판을 이렇게 구성해보세요"** 라는 구체적인 실행 방안을 제안함.
- **감정 변화:** '도움이 된다' → '행동이 가능해졌다'.

### [Act 4: 신뢰] - "나를 믿고 달라요"
- **상황:** 첫 달 매출이 전년 대비 20% 증가한 결과를 확인함.
- **기술적 접점 (Trust Gauge):** "오늘의 안정성 지표 92%"라는 Trust Widget 이 화면 상단에 지속적으로 표시되며, "안정적으로 성장하고 있구나"라는 심리적 안정감 제공.
- **클라이맥스:** "BDS 는 데이터를 분석해 주는 기계가 아니라, 당신의 가게를 위한 든든한 파트너입니다."라는 메시지 전달.

## 3. 마케팅/교육 자료 개요 (Content Matrix)

### [A] 인스타그램 캠페인: "#내가게의안심"
- **포맷:** 릴스 (Reels), 피드 이미지
- **핵심 메시지:** "불안을 데이터로 해결하세요."
- **콘텐츠 구조:**
  - **후크 (Hook):** "당신의 가게 매출, 정말 불안하십니까?" → 시각적 자극 (어두운 방 vs 밝은 플랫폼 UI)
  - **본문 (Body):** Trust Widget/PainGauge 을 활용한 실제 사용자 경험 사례 3 분 릴스.
    - "이 도구를 사용하면 이런 문제가 해결됩니다."
    - "기술적 용어 없이, 당신의 언어로 설명드립니다."
  - **CTA:** "오늘부터 당신의 가게를 더 잘 알기 시작하세요."

### [B] 블로그 글: "불안에서 신뢰까지, 데이터 기반 성장 여정"
- **제목:** "소상공인 위한 감정 관리 가이드: 불안감을 줄이는 3 가지 데이터 활용법"
- **목차:**
  1.  소상공인의 불안감은 왜 생길까요? (Pain Point 분석)
  2.  BDS 플랫폼이 제안하는 해결책 (Trust Widget/PainGauge 소개)
  3.  실제 사례: A 사장님의 이야기 (스토리텔링 적용)
  4.  지금 바로 시작하기

### [C] 교육 자료: "BDS 데이터 활용 워크숍 커리큘럼"
- **대상:** 소상공인, 지역 상공회 회원
- **목표:** 플랫폼의 기능을 이해하고, 이를 활용해 실제 비즈니스를 개선하는 법을 배우게 됨.
- **커리큘럼 개요:**
  - **Module 1: 불안 감수성 이해하기** (데이터가 어떻게 감정과 연결되는지 설명)
  - **Module 2: Trust Widget/PainGauge 실습** (인터랙티브 UI 를 직접 조작하며 경험)
  - **Module 3: 데이터 기반 의사결정 사례 연구** (실제 매출 데이터를 활용한 전략 수립)
