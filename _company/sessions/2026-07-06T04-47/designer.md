| **Primary Tone** | 신뢰, 과학적 증명, 해결책 제시 | `#007bff` (신뢰 블루), `#28a745` (성장 녹색) | Inter (Bold/Semi-Bold for impact) |

### 2. Shot List 기반 세그먼트별 상세 레이아웃 지침

다음은 `youtube.md`의 Shot List를 기반으로, `final_video_production_brief.md`에서 정의한 내러티브 흐름에 따라 시각적 연출을 구체화한 결과입니다.

| 시간대 | 내레이션/스크립트 핵심 | 비주얼 연출 지침 (Designer 반영) | 시각적 강조 요소 (Pain/Gain Gauge) |
| :---: | :--- | :--- | :--- |
| **0:00-0:15** | **[강렬 후크]** "매출이 안 오를 때... '그냥 더 열심히' 하셨죠? 하지만 그건 근본적인 해결책이 아닙니다." | **스타일:** 극도의 불안감 연출. 🟥 빨간색 필터(Overlay) 적용. 사장님의 깊은 고민과 불안감을 극대화하는 클로즈업 (Face/Hand). 배경은 어둡고 대비가 강하게 처리. | **Pain Gauge:** 최저점 (`#dc3545` 사용). 화면 전체에 경고 색상 오버레이. |
| **0:15-1:30** | **[문제 공감/진단]** "사장님, 지금 매출을 손으로 계산만 하고 계시진 않나요? 문제는 당신의 가게가 매번 불안한 예측 속에 놓여 있다는 겁니다." | **스타일:** 대비 강조. 손으로 수기로 계산하는 모습과 복잡하게 얽힌 데이터 대시보드 화면의 극적인 대비 연출 (Split Screen 또는 Quick Cut). 시각적으로 혼란스러움을 표현. | **Pain Gauge:** 중간 단계 (`#ffc107` 노랑)로 전환되며, 불안정성이 시각화됨. |
| **1:30-3:00** | **[핵심 이론/지식 자본화]** "진짜 문제는 데이터를 활용하지 못하는 겁니다. 성공은 '감'이 아니라, 데이터로 만드는 '안전 마진'에서 시작합니다." (공식 1, 2, 3 제시) | **스타일:** 전환 및 해결책 제시. 화면 전체에 신뢰 블루(`#007bff`)와 성장 녹색(`#28a745`)의 조합을 사용하여 전문성과 희망감을 동시에 전달. 데이터 시각화(차트/그래프)가 명확하고 깔끔하게 제시됨. | **Gain Gauge:** 목표 지점 (`#28a745` 사용). '안전 마진' 개념이 그래프나 도형으로 명확히 시각화되어 강조됨. |
| **3:00-End** | **[행동 촉구/CTA]** "지금 바로 데이터 기반의 예측 시스템을 통해, 불확실성을 제거하고 안정적인 성장의 길로 나아가십시오." | **스타일:** 깔끔함과 자신감. 핵심 솔루션(플랫폼) UI 목업이나 최종 결과 화면을 중앙에 배치. CTA 버튼은 신뢰 블루(`#007bff`)와 녹색의 대비를 통해 시선을 집중시킴. | **Trust Widget/PainGauge:** 최적화된 상태 (Green/High Trust). 플랫폼 로고 및 명확한 CTA 강조. |

### 3. 디자인 적용 원칙 (Application Rules)

1.  **일관성 유지:** 모든 데이터 시각화(차트, 게이지)는 반드시 **`#007bff`**와 **`#28a745`** 색상 팔레트를 사용하여 브랜드의 '신뢰'와 '성장'이라는 핵심 가치를 일관되게 반영해야 합니다.
2.  **감정 유도:** 초기 구간(Pain)에서는 강한 대비와 빨간색 계열을, 해결책 제시 구간(Gain)에서는 명확하고 안정적인 블루/그린 톤을 사용하여 시청자의 감정 변화를 즉각적으로 유도합니다.
3.  **정보 밀도 조절:** 핵심 이론 설명 부분은 복잡한 데이터 대신 **단일 개념과 강력한 비유**를 시각화하여 정보의 과부하를 막고, 최종 CTA 구간에서는 플랫폼의 **명확한 UI 목업**을 통해 솔루션의 실체를 제시합니다.
