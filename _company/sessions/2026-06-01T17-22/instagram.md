## 🎯 콘텐츠 분포 계획 (Writer 의 5 종 영상 기반)

| 순서 | Writer 영상 주제 | 최적 형식 | 이유 및 전략적 고려사항 |
| :--- | :--- | :--- | :--- |
| 1️⃣ | **현금 흐름의 공포를 끝내세요**<br>(Pain: 카드값 스트레스 → Solution: 실시간 정산) | **릴스 (Reels)** | - 후크 0~3 초의 강렬한 시각적 자극 (빨간색 차트 흔들림)<br>- 숏폼 플랫폼에서 가장 높은 도달률 보장<br>- CTA 로 앱 스토어 직접 연결 가능 |
| 2️⃣ | **고객 확보, 이제 AI 가 해줘요**<br>(Pain: 광고비 증가 → Solution: AI 마케팅 도구) | **릴스 (Reels)** | - "광고비만 더 많이 들어요?"라는 텍스트 오버레이가 숏폼에 최적<br>- AI 도구를 화면에서 직접 보여줌으로써 신뢰도 상승 |
| 3️⃣ | **유통 허브, 안정적 수익 모델**<br>(Pain: 공급망 단절 → Solution: 중간 유통 허브) | **카로셀 피드 (Carousel)** | - 플랫폼 기능 설명은 텍스트 기반의 교육 콘텐츠가 더 효과적<br>- 3~4 장 슬라이드로 Pain Point → Solution → Case Study 순서 배치<br>- 사용자가 스크롤하며 자연스럽게 학습하는 구조 |
| 4️⃣ | **AI 마케팅 자동화 실전 가이드**<br>(Pain: 시간 부족 → Solution: 일괄 처리 기능) | **릴스 + 피드 조합** | - 릴스: 실제 화면 녹화 (스크립트 기반 빠른 편집)<br>- 피드: 가이드북 스타일의 카로셀 ("1 단계", "2 단계", "결과")<br>- 크로스 플랫폼 전략으로 도달 극대화 |
| 5️⃣ | **초기 혜택 및 성공 사례**<br>(Pain: 자본 부족 → Solution: 초기 할인, 커뮤니티) | **피드 (단일 이미지 + 장문의 캡션)** | - 구체적인 숫자 (할인율, 가입자 수 증가율) 를 시각적으로 강조<br>- 스토리와 함께 게시하여 인게이지먼트 유도<br>- 해시태그 최적화로 검색 노출 지속성 확보 |

## 📊 최적 게시 시간 전략 (Korean Business Owners 기준)

**총칙:** 한국 소상공인의 스마트폰 사용 패턴 (출퇴근, 점심시간, 퇴근 후) 을 고려한 3 개 시점 배치

| 시점 | 시간대 | 목표 및 콘텐츠 타입 |
| :--- | :--- | :--- |
| **아침 피크** | 07:30 ~ 08:30 | - 출퇴근 중 스마트폰 확인<br>- **릴스 #1, #2 (현금 흐름/광고비)**<br>→ "오늘 일할 에너지"를 주는 긍정적 메시지 |
| **점심 휴식** | 11:45 ~ 12:45 | - 업무 중간 휴식 시간<br>- **카로셀 #3 (유통 허브) + 피드 #5 (성공 사례)**<br>→ "쉬면서 배우는 교육 콘텐츠" |
| **저녁 피크** | 18:00 ~ 20:00 | - 퇴근 후 스트레스 해소 및 정보 탐색<br>- **릴스 #4 (AI 가이드) + 피드 #5 (성공 사례)**<br>→ "나를 위한 투자" 느낌을 주는 가치 전달 |

**주의:** 
- 모든 게시물은 **하루에 1~2 회 이상 반복**되어야 합니다.
- 릴스는 아침/저녁 시점에 집중, 카로셀은 점심과 저녁 피크에 배치합니다.
- **스토리 (Story)**는 아침 07:00 과 오후 15:00 에 "오늘의 팁"으로 발행하여 인게이지먼트 유지해야 합니다.

## 🎨 시각 에셋 준비 및 제작 요구사항

### 1️⃣ 릴스 전용 (Reels Assets)
| 요소 | 요구사항 | 디자인 토큰 |
| :--- | :--- | :--- |
| **후크 오버레이** | 0~3 초 텍스트 (빨간색 → 초록색 전환) | `Pain_Red: #D62828`, `Solution_Green: #28A745` |
| **백그라운드** | 플랫폼 대시보드 화면 녹화 또는 고해상도 스테오그래피 이미지 | `BG_Dark: #1F2937`, `Accent_Blue: #3B82F6` |
| **자막** | 흰색 텍스트 + 검은 그림자 (가독성 확보) | `Font: Pretendard Bold 40px`, `Shadow: black 2px 2px 0` |
| **CTA 버튼** | "지금 가입하세요" 애니메이션 (플리팅 효과) | `Button_Green: #16A34A`, `White_Text: #FFFFFF` |

### 2️⃣ 피드 전용 (Feed Assets)
| 요소 | 요구사항 | 디자인 토큰 |
| :--- | :--- | :--- |
| **카로셀 타이틀** | 각 슬라이드 헤더에 브랜드 로고 + 하이라이트 텍스트 | `Header_Dark: #111827`, `Highlight_Blue: #3B82F6` |
| **성공 사례 이미지** | 실제 소상공인 프로필 (퍼블릭 도메인) 또는 일러스트레이션 | `Human_Tone: Warm_Brown`, `Success_Green: #10B981` |
| **데이터 시각화** | 차트/그래프는 플랫폼 내부 대시보드 스크린샷 활용 | `Chart_Red: #EF4444`, `Chart_Blue: #6366F1` |

### 3️⃣ 공통 요구사항 (All Assets)
- **브랜딩:** 모든 에셋에 하단에 BDS 플랫폼 로고 워터마크 포함  
- **가독성:** 한국어 폰트 (Pretendard, Noto Sans KR)만 사용
