## 🛠️ Recommended Action Plan Card 디자인 핸드오프 사양 (V1.0)

### 1. 컴포넌트 개요 및 역할 정의
*   **컴포넌트명**: `ActionPlanCard`
*   **위치**: KPI Gauge의 우측 또는 하단에 배치되는 보조 정보 블록.
*   **목표**: 사용자의 현재 상태(Gauge)를 바탕으로, 가장 시급하고 실행 가능한 3단계 액션 로드맵을 제공하여 불안감을 '구체적인 할 일 목록'으로 치환한다.
*   **핵심 원칙**: 절대 모호한 조언이나 거대한 목표를 제시하지 않는다. **"다음 24시간 동안 무엇을 해야 하는가?"**에 초점을 맞춘다.

### 2. 와이어프레임 및 레이아웃 구조 (Desktop/Mobile)
(개발팀 핸드오프용 상세 스케치 정의)

| 영역 | 요소명 | 내용 구성 | 크기 비율 (W:H) | 상호작용/토큰 |
| :--- | :--- | :--- | :--- | :--- |
| **A. 상태 헤더** | `State Title` | Gauge의 현재 감정 상태를 명시적이고 공감적으로 재진술한다. (예: "🚨 지금은 주의가 필요합니다.") | 100% / 80px | 배경색/텍스트 색상: Gauge와 연동 (`Red`/`Yellow`/`Green`) |
| **B. 핵심 메시지** | `Empathy Headline` | 소상공인의 불안감을 인정하고 공감하는 문장으로 시작한다. (예: "너무 막막하게 느껴지시죠? 걱정 마세요.") | 100% / 50px | 폰트: Body Text, 크기 조정(Emphasis) |
| **C. 액션 로드맵** | `3-Step Plan` | 3단계의 구조화된 해결 방안을 제시한다. 각 단계는 아이콘과 짧은 문구로 구성되어야 한다. | 100% / N * 120px | 토큰: Step Marker (순차성 강조), Hover Effect (클릭 유도) |
| **D. 최종 CTA** | `Primary Action Button` | 가장 먼저, 그리고 반드시 수행해야 할 단일 액션 버튼을 제시한다. | 100% / 60px | 디자인: Primary Button 토큰 사용 (`BDS-Blue`) |

### 3. 시각적 디자인 토큰 정의 (Design Tokens)

| 토큰 이름 | 역할 및 의미 | 값 (Value) | 적용 영역 | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **`--color-state-red`** | 비상/위험 (Critical) | `#E74C3C` | A. State Header 배경, 텍스트 하이라이트 | 긴급성, 즉각적 주목 유도 |
| **`--color-state-yellow`** | 주의/경고 (Warning) | `#F39C12` | A. State Header 배경, 텍스트 하이라이트 | 신중함, 경계심 유도 |
| **`--color-state-green`** | 안정/성공 (Normal) | `#2ECC71` | A. State Header 배경, 텍스트 하이라이트 | 안도감, 지속 가능성을 의미 |
| **`--bg-card`** | 카드 기본 배경 | `#FFFFFF` | Card 전체 배경 | 깨끗하고 집중도를 높임 |

| **`--button-primary-bg`** | 핵심 CTA 버튼 배경 | `#007AFF` | D. Primary Action Button | BDS 브랜드의 주력 컬러 사용 (일관성 유지) |

### 4. 상태별 콘텐츠 예시 및 개발 가이드라인

#### 🔴 Critical State (위험/비상 상황)
*   **A. State Title**: **"⚠️ 비상 경고! 지금 즉각적인 진단이 필요합니다."** (`--color-state-red` 배경)
*   **B. Empathy Headline**: "너무 불안해서 뭘 해야 할지 모르시죠? 일단 멈추세요. 가장 먼저 전문가의 도움을 받아야 합니다."
*   **C. 액션 로드맵 (3단계)**:
    1.  **전문가 연결:** [전화/채팅 아이콘] - 즉시 무료 진단 서비스를 신청하세요. (최우선)
    2.  **위험 요인 파악:** [돋보기 아이콘] - 최근 3개월간의 핵심 매출 데이터와 비용 구조를 재점검합니다.
    3.  **지원 제도 확인:** [체크리스트 아이콘] - 정부 및 지자체의 긴급 지원 제도를 찾아보세요. (정보 습득)
*   **D. Primary CTA**: **"전문가 무료 진단 신청하기"** (최우선 버튼 활성화)

#### 🟡 Warning State (주의/경고 상황)
*   **A. State Title**: **"⚠️ 주의 필요! 작은 점검이 큰 변화를 만듭니다."** (`--color-state-yellow` 배경)
*   **B. Empathy Headline**: "괜찮다고 넘기기엔 불안한 느낌, 공감합니다. 아직 해결할 시간이 충분해요. 가장 쉬운 것부터 시작해 봅시다."
*   **C. 액션 로드맵 (3단계)**:
    1.  **원인 분석:** [분석 아이콘] - 매출 하락/지출 증가의 근본 원인을 2가지로 압축합니다.
    2.  **가설 설정:** [전구 아이콘] - 가장 가능성이 높은 해결책을 세우고 작은 실험부터 시작할 준비를 합니다.
    3.  **피드백 수집:** [말풍선 아이콘] - 주변 소상공인이나 지인을 통해 시장의 반응을 객관적으로 확인합니다.
*   **D. Primary CTA**: **"무료 컨설팅 예약하기 (1:1 심층 분석)"**

#### 🟢 Normal State (안정/성장 추구 상황)
*   **A. State Title**: **"✅ 안정적입니다! 다음 단계 성장을 계획할 차례예요."** (`--color-state-green` 배경)
*   **B. Empathy Headline**: "지금 잘하고 계세요. 이 좋은 흐름을 유지하며, '다음 레벨'로 올라갈 준비를 해봅시다."
*   **C. 액션 로드맵 (3단계)**:
    1.  **성장 기회 포착:** [돋보기 아이콘] - 새로운 시장 트렌드를 학습하고 잠재적 수요처를 발굴합니다.
    2.  **효율 극대화:** [톱니바퀴 아이콘] - 현재 운영 중인 프로세스에서 비효율적인 부분을 찾아 개선할 방법을 모색합니다.
    3.  **시스템 확장:** [연결 고리 아이콘] - 파트너십이나 지역 커뮤니티와 연결하여 사업 범위를 넓힙니다.
*   **D. Primary CTA**: **"성장 가이드 콘텐츠 확인하기"**
