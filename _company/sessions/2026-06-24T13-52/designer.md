### 🖼️ Instagram 콘텐츠 최적화 비주얼 에셋 제작 (Risk Index Dashboard 기반)

다음은 'Risk Index Dashboard' 데이터를 시각적으로 압축하고, 플랫폼의 신뢰도와 긴급성을 전달하는 데 초점을 맞춘 3가지 컨셉과 포맷입니다.

#### **컨셉 1: 위험 지표 강조형 (The Urgent Call)**
*   **목표:** 즉각적인 경고 및 행동 유도. 시스템 안정성 데이터와 위험도를 직접적으로 연결하여 긴급성을 극대화합니다.
*   **레이아웃:** 세로형 (4:5 또는 9:16)
*   **주요 색상 사용:** Deep Blue (배경/신뢰), Growth Green (위험 레벨 강조).
*   **비주얼 요소:** 중앙에 가장 위험한 지표(Risk Score)를 크고 굵게 배치하고, 주변에 녹색/빨간색으로 위험도 레벨을 명확히 구분합니다.

| 항목 | 상세 디자인 사양 | 적용 색상 코드 |
| :--- | :--- | :--- |
| **포맷** | Instagram Story / Reel Cover (9:16) | N/A |
| **배경** | Deep Blue (#004D66) 그라데이션 | `#004D66` $\rightarrow$ `#1A5365` |
| **핵심 메시지** | "당신의 AI 매출 위험 점수는? 즉시 확인하세요." (굵은 타이포) | 흰색 (`#FFFFFF`) |
| **데이터 시각화** | 리스크 레벨(A/B/C 등)을 원형 차트 또는 게이지 형태로 중앙에 배치. 위험도가 높을수록 Growth Green의 채도를 높입니다. | `#3CB371` (High Risk), `#82B3A9` (Medium Risk) |
| **CTA 영역** | 하단에 "전략 가이드 받기" 버튼 (Growth Green 배경, Deep Blue 텍스트). | `#3CB371` / `#004D66` |

#### **컨셉 2: 성장 로드맵 강조형 (The Growth Path)**
*   **목표:** 데이터 기반의 해결책과 미래 성장의 가능성을 제시하여 플랫폼 사용의 이점을 강조합니다.
*   **레이아웃:** 정사각형 또는 세로형 (1:1 / 4:5)
*   **주요 색상 사용:** Growth Green (성장), Deep Blue (기반).
*   **비주얼 요소:** 좌측에 현재 상태(Risk Index)를 보여주고, 우측에는 해결책(Action Plan/로드맵)을 화살표 또는 단계별 흐름도로 시각화합니다.

| 항목 | 상세 디자인 사양 | 적용 색상 코드 |
| :--- | :--- | :--- |
| **포맷** | Instagram Post / Carousel Card (1:1) | N/A |
| **배경** | 밝은 회색 계열 또는 소프트한 Deep Blue 배경. | `#F5F5F5` |
| **핵심 메시지** | "위험 진단부터 성장 전략까지, AI가 제시하는 명확한 로드맵." | 검정/짙은 파랑 |
| **데이터 시각화** | 좌측에 '현재 위험 레벨'을 강조하고, 우측에는 단계별 목표(Step 1: 진단 $\rightarrow$ Step 2: 대응 $\rightarrow$ Step 3: 성장)를 아이콘과 함께 배치. | Growth Green으로 각 단계를 연결하는 화살표 사용. |
| **CTA 영역** | "전체 로드맵 보기" (Deep Blue 버튼). | `#004D66` |

#### **컨셉 3: 데이터 신뢰도 강조형 (The Trust Builder)**
*   **목표:** 플랫폼의 기술적 신뢰성과 데이터 기반 컨설팅의 전문성을 극대화합니다.
*   **레이아웃:** 세로형 (9:16) 또는 넓은 배너 형식.
*   **주요 색상 사용:** Deep Blue (신뢰), 화이트/라이트 그레이 (클린함).
*   **비주얼 요소:** 복잡한 데이터(차트, 지표)를 깔끔하게 정리하여 'AI가 분석했다'는 느낌을 강조합니다.

| 항목 | 상세 디자인 사양 | 적용 색상 코드 |
| :--- | :--- | :--- |
| **포맷** | Instagram Reel Cover / Infographic Style (9:16) | N/A |
| **배경** | 깨끗한 화이트 또는 매우 옅은 Deep Blue. | `#FFFFFF` |
| **핵심 메시지** | "시스템 안정성 데이터로 예측하는 소상공인 AI 매출 위험도" | 진한 네이비 색상 |
| **데이터 시각화** | Risk Index Dashboard의 핵심 차트(예: Uptime, MTTR 등)를 깔끔한 라인 그래프 형태로 배치하고, 가장 중요한 위험 지표는 Deep Blue로 강조합니다. | `#004D66` 및 회색 계열 |
| **CTA 영역** | "전문 리포트 요청하기" (Deep Blue 버튼). | `#004D66` |

### 📋 디자인 가이드라인 요약 (Designer System)

이 세 가지 컨셉을 기반으로, 모든 Instagram 콘텐츠는 다음의 비주얼 시스템을 따릅니다.

1.  **Primary Color:** Deep Blue (`#004D66`) - 신뢰, 안정성, 전문성
2.  **Accent Color:** Growth Green (`#3CB371`) - 성장, 긍정적 변화, 행동 유도
3.  **Typography:** 산세리프 계열 (예: Pretendard 또는 Noto Sans)을 사용하여 데이터의 명확성을 확보합니다. 핵심 지표는 볼드체로 강조합니다.
4.  **레이아웃 원칙:** 정보의 위계(Hierarchy)를 명확히 하여, 사용자가 스크롤하는 즉시 가장 중요한 '위험도'와 '해결책'을 인지하도록 설계합니다. 모든 데이터 시각화는 차트/게이지 형식을 선호하며, 텍스트 설명은 최소화하고 색상과 도형으로 의미를 전달합니다.
