# 🛡️ BDS 소상공인 플랫폼 (sosang-platform)

BDS 소상공인 플랫폼은 기술적 안정성을 사용자에게 'Proof of Trust(신뢰의 증명)'로 시각화하여, 서비스 신뢰도를 극대화하는 소상공인 맞춤형 플랫폼입니다.

---

## 🚀 실행 안내 (Getting Started)

### ⚠️ 첫 실행 시 주의사항 (코드 서명 전)
윈도우용 설치 파일인 **`Connect-AI-0.1.0-win-x64-Setup.exe`** 등 앱 패키징 후 최초 실행 시, 운영체제 보안 정책(코드 서명 미적용)으로 인해 **"확인되지 않은 개발자"** 경고 또는 SmartScreen 경고가 발생할 수 있습니다. 아래의 단계를 따라 실행해 주세요.

*   **윈도우 (Windows):**
    1. 경고 창에서 **[추가 정보]** 링크를 클릭합니다.
    2. 활성화된 **[실행]** 버튼을 클릭하여 앱을 시작합니다.
*   **맥OS (macOS):**
    1. 앱 실행 시 경고 창이 뜨면 확인을 누릅니다.
    2. **[시스템 설정] → [개인정보 보호 및 보안]**으로 이동합니다.
    3. 하단의 안전성 관련 항목에서 **[확인 없이 열기]**를 클릭합니다.
